# 比巴卜作手观察画像

## 当前观察

- 作者的复盘顺序稳定为“指数与量能 → 情绪周期 → 题材载体 → 板块内部角色 → 核心票反馈 → 仓位与次日预案”。单票观点通常服从市场环境和题材节奏，不单独脱离周期讨论。
- 作者把短线交易理解为情绪、资金、概率和赔率的组合：先判断市场处在混沌、主升还是退潮，再决定该试错、加仓核心、卖高潮，还是空仓防守。
- “情绪只是外在，题材才是载体”是课程和每日复盘共同出现的核心框架。判断题材持续性时，重点看核心主动性、后排健康度、中军或容量承接、分歧后回流质量，而不是只看涨停数量或当天涨幅。
- 作者很重视板块内角色定位，会区分龙头、情绪票、身位票、中军、容量票、弹性票、趋势核心、跟风、补涨和后排杂毛。普通交易者不必只等最终龙头确认，而是先把绕不开的核心角色放进观察池，再等待市场选择。
- 资金风格分类是重要前置判断。机构趋势、量化轮动、情绪连板、消息套利、高低切和避险抱团对应不同买点、持有周期和仓位，不能用同一套追涨或打板逻辑处理。
- 作者偏好“核心票 + 条件预案”，尤其重视分歧后的核心承接和主动回流；一致高潮、消息首日过度一致、尾盘抢筹透支次日需求时，通常会降低后手追高意愿。
- 技术分析是辅助工具而非唯一依据。常用均线、缺口、支撑压力、箱体、趋势线、量价、CCI、三日止盈、平台底部和五日线纪律，但会把这些工具放到题材、情绪、资金风格和监管约束中综合使用。
- 仓位管理随市场容错率和账户反馈动态变化。主升期不能轻易空仓，退潮期或没有持续方向时休息是策略；月底、节前、重大事件前，会用五至七成、小仓试错或主动降仓来处理不确定性。
- 作者会复盘公开预案与实际操作，允许预案被公告、外围、期货、竞价、盘中反馈推翻；若核心补跌、低位承接失败、竞争失败或自身模式连续亏钱，会把退出和降仓作为计划的一部分。

## 已沉淀的方法框架

- **复盘工作流**：大盘技术与消息面、市场量能、情绪高标与断板反馈、板块节奏、核心票池、涨停原因、持仓预期和止盈止损。
- **盘中观察链**：竞价一字方向和封单变化、开盘自然涨停、板块涨幅榜与涨速榜、短线精灵、三分钟涨跌速、核心票和后排反馈、板块对流与互相吸血。
- **情绪周期**：混沌期小仓低位试错，主升期重视主线核心和分歧买点，退潮期空仓或极小仓博弈反弹；量化时代小周期加速后，要放弃很多小分歧节点。
- **题材节奏**：启动、高潮、分化、强弱分歧、强弱回流、连续分歧修复、尾盘抢筹和一致回避。分歧不等于结束，核心批量破位和无法回流才更接近结束。
- **择股顺序**：先择时和主流题材，再在题材内找核心票；用领涨、带动、扛跌判断龙头三性，再结合区间涨幅、竞价、涨速、赚钱效应和逻辑研究筛选。
- **具体战法**：弱转强、监管套利、事件埋伏、机构趋势三日止盈、老庄趋势和一字定方向都不是孤立公式，必须先满足环境、题材、角色、量价和赔率条件。
- **风险处理**：高潮次日去弱留强，找不到持续方向则休息；高低切必须证明低位能承接赚钱效应，不能因为高位涨多就机械切低位。

## 待长期验证

- CCI 极值信号在不同市场环境下的胜率和盈亏比。
- “穿越冰点”题材成为后续主线的可靠程度。
- 机构、量化、情绪题材分类是否能稳定指导仓位和持有周期。
- 公开计划与次日实际走势之间的命中率。
- 强弱分歧、强弱回流、尾盘抢筹和一致高潮对次日溢价的统计效果。
- 弱转强、事件埋伏、监管套利、机构趋势三日止盈等具体战法在不同周期下的胜率、赔率和失效边界。
- 低位高低切能否成立，是否能由连续强度、核心角色、赚钱扩散和次日承接形成可量化确认。
- 作者用账户反馈调整仓位的规则，能否长期减少回撤并保留主升期收益。

## 更新规则

每分析一期视频，记录日期、关键判断、触发条件、失效条件、次日结果、对应情绪周期、题材角色、核心票反馈和仓位动作。至少积累 20 期后，再总结稳定交易准则与常见偏差；课程中已经反复确认的方法可以先写入“已沉淀的方法框架”，但单期观点必须继续保留为待验证样本。
