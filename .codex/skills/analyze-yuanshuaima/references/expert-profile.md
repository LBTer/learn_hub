# 袁帅马观察画像

## 当前观察

- 当前样本显示，作者的普通公开视频更接近“个股趋势分析 / 主力操盘讲解”，不是标准交易日复盘。
- 作者会围绕某只核心个股的历史走势做详细解析，再抽取部分个股简单讲后续可以如何参与。
- 第一条样本《 中国巨石主力操盘讲解 》标题指向单股主力操盘讲解，说明后续分析应重点关注趋势阶段、主力行为、图表位置和参与条件。
- 2026-06-25《 中国巨石主力操盘讲解 》中，作者先用中国巨石完整拆解“杀跌后箱体 -> 套牢盘清洗 -> 假头部 -> 缺口保护 -> 洗盘/对倒 -> 换人 -> 三日线与五日线逼空”，再快速点评评论区个股。
- 该期反复出现的参与语言是“找趋势”“找支撑”“等回踩”“等止跌”“等阴线修复”“等突破回踩确认”，说明其普通日常分析更偏条件参与，而不是直接给确定买点。
- 该期明确把“洗盘 K 线很吵、又急又乱”和“出货通常更缓和地拉高派发”作区分，但仍需多期验证是否为稳定框架。
- 2026-06-29《凯盛科技主力操盘讲解》中，作者把“长时间区间震荡/主力操盘”和“消息面支持”放在一起看，并用温和放量、小趋势、五日线回踩来判断直上直下走势是否转为趋势性拉升。
- 当前样本数量不足，不应过早归纳稳定交易系统、胜率或固定风格。

## 需要持续观察的维度

- **视频类型**：普通日常分析、单股深度、会员合集、专题教学、问答、复盘或其他。
- **讲述顺序**：是否稳定按“历史走势 -> 主力行为 -> 当前阶段 -> 后续参与 -> 风险”展开。
- **核心工具**：均线、趋势线、成交量、筹码、K 线形态、分时盘口、压力支撑、平台结构、缺口、指标或其他。
- **趋势阶段划分**：作者如何定义底部、吸筹、洗盘、试盘、突破、主升、震荡、出货和破位。
- **主力行为术语**：吸筹、洗盘、压盘、拉升、试盘、诱多、出货、控盘、震仓等词在作者体系中的含义。
- **参与条件**：作者给出买点时更重视回踩、突破、放量、缩量、均线、平台、分时确认还是资金情绪。
- **风险边界**：是否给出止损、失效、不能追高、等待确认、仓位控制和适用行情。
- **简要个股处理**：作者如何筛选附带讲解的股票，是否与核心案例共享同一方法。
- **新手教学**：是否会解释基础图形、主力思路、参与纪律和常见误区。
- **事后验证**：后续视频是否回看此前判断，如何处理失败案例或条件未触发。

## 待沉淀的方法框架

以下内容需要至少多期验证后再写入稳定框架：

- 袁帅马的趋势阶段模型。
- 主力操盘路径的常用判断标准。
- 核心个股与简要个股的筛选逻辑。
- 后续参与条件与失效条件的固定表达。
- 会员合集中的课程体系和主题分类。
- 作者对失败、破位、不及预期和仓位处理的规则。

## 更新规则

每分析一期视频，记录：

- 日期、BVID、标题和视频类型。
- 核心股票、辅助股票和简要个股。
- 作者划分的趋势阶段。
- 关键图表证据。
- 后续参与条件和失效条件。
- 本期出现的新术语、新方法或新合集名称。
- 后续实际走势或作者复盘时的验证结果。

至少积累 10 至 20 期后，再总结稳定交易准则、常用图表工具、预测偏差和可迁移方法。单期观点只能作为待验证样本，不直接写成长期结论。
