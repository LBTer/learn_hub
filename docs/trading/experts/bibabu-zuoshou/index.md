# 比巴卜作手

这里整理 B 站 UP 主“比巴卜作手”的交易思路、每日复盘、充电会员复盘、体系课程和技术点学习。

> 页面内容是对视频的学习整理，不构成投资建议。视频中的判断具有时效性，应结合后续走势验证。

## 当前认知

比巴卜作手的核心框架不是单纯找强势股，而是先判断市场环境，再把题材、资金风格、板块角色和个股反馈放到同一张图里。每日复盘通常从指数、量能和情绪开始，再进入题材强弱、核心票承接、后排反馈、仓位和次日预案。

目前文档中已经沉淀出的稳定认知：

- **周期优先**：先区分混沌、主升、退潮。混沌期偏小仓试错，主升期围绕主线核心和分歧买点，退潮期以空仓、防守或极小仓博弈为主。
- **题材是载体**：涨停数量不等于持续性。更关键的是核心能否主动、后排是否健康、中军或容量票能否承接、分歧后是否强回流。
- **核心票分层**：作者会区分龙头、情绪票、身位票、中军、容量票、弹性票、趋势核心、跟风、补涨和后排杂毛。真正的交易机会通常来自市场对这些角色的再选择。
- **风格匹配模式**：机构趋势、量化轮动、情绪连板、消息套利、高低切和避险抱团，对应不同买点、持有周期和仓位。不能把所有上涨都套进同一种追涨逻辑。
- **分歧重于高潮**：一致高潮、消息首日过度一致、尾盘抢筹透支次日需求时，后手赔率通常下降；分歧后的核心承接和主动回流更有观察价值。
- **技术工具辅助确认**：均线、缺口、支撑压力、箱体、趋势线、量价、CCI、三日止盈和五日线纪律都常出现，但它们要和题材、情绪、资金风格、监管空间一起使用。
- **预案必须可证伪**：作者经常给出触发条件、失效条件和仓位动作。公告、外围、期货、竞价和盘中反馈都可以推翻前一晚预案；竞争失败、核心补跌或模式连续失效时，应及时退出或降仓。

后续更新人物认知时，优先把新视频放回这些框架里验证：它是在补充周期判断、题材节奏、角色定位、具体战法，还是在修正某个失效条件。
