## 全片时间轴

### 00:00 课程开场：用小周期择时定义买点和卖点

作者开场说，本期讲“游资择时方法”里的小周期择时。学完这一课，核心目标不是多记几个股票，而是通过小周期来定义自己的操作，选择更好的买点和卖点。

他先回到前面情绪周期课程讲过的大周期：混沌、主升、退潮。大周期先定方向，小周期再定具体买卖位置。

![大周期与小周期框架](https://learn.luovista.com/trading/bibabu/member-hot-money-system/11/0020.jpg)

**视频时间点：00:20**

### 00:55 大周期定义：有周期龙是主升，没有周期龙是混沌，周期龙 A 杀是退潮

作者用金属板块举例。白银有色连板向上顶，湖南白银等共振时，就是有周期龙、进入主升。之后周期龙跌停开始退潮，退潮几天后如果没有市场公认龙头，就又回到混沌期。

他强调，题材节奏表最上方每天都会写当前情绪位置，比如金属主升、金属退潮、混沌等。大周期本身并不难定义：有市场公认龙头就是主升，没有龙头就是混沌，周期龙 A 杀就是退潮。

### 02:40 混沌期的小周期：回流、高潮、分歧、分歧延续，再博弈回流

确定当前大周期是混沌后，再看混沌期的小周期节奏。作者给出的节奏是：先回流，再高潮；高潮主卖；卖完之后等分歧和分歧延续，通常一到两天之后再博弈回流。

他提醒，混沌期的强回流往往就是小周期高潮。题材节奏表里从周三、周四开始的强回流板块，第二天大概率会接分歧，很多时候 80% 以上都会分歧。高潮日卖掉，多数情况下并不亏，因为第二天除了最核心的票可能继续转强，其他票很容易接分歧。

![混沌期回流高潮后接分歧](https://learn.luovista.com/trading/bibabu/member-hot-money-system/11/0170.jpg)

**视频时间点：02:50**

作者举了几个板块节奏：AI 玩具在周四中回流，周五表现很差；光伏某次强回流后虽然第二天延续，但这是混沌期里较偶然的板块，延续后也很快全线下杀；云计算弱回流后也直接反杀；消费周四强回流，周五也接反杀。周五真正强的是 CPO、电池、光伏等已经分歧或大分歧后的板块。

因此混沌期的操作不是追高潮，而是买在分歧节点，尤其是连续分歧后的回流预期。

### 05:50 主升期的小周期：仓位比个股重要，分歧给买点

作者转到主升期。他说主升期和混沌期完全不同，主升期的核心理念是仓位比个股重要，要的是贪婪和头铁。主升期的短线节奏通常是回流后分化或小分歧，再高潮；高潮后分化分歧，很快又接回流和高潮。

在主升期，小周期择时没有混沌期那么重要，因为市场本身强，“你干就能赚钱”。关键是仓位，只要买到主线票，大多能赚钱。主升期的操作可以简单理解为：给分歧点就买。

### 06:50 退潮期的小周期：所有买点都可能是套人的点

退潮期最恐怖。作者说，退潮期的小周期是分歧、分歧延续、再接回流，然后又分歧延续。很多你以为是买点的位置，其实都是套人的位置。

他用白银有色举例。退潮里的回流日，如果能在地板或低位博弈，日内可能赚十个点，第二天冲高还能带出来几个点。但这要求节点非常完美，难度极高。退潮期只有一个相对可做的点：分歧延续后的回流；其他位置基本都偏卖点。

### 08:05 三类环境的节奏对照

作者把三类环境重新压缩成几句话：

- 混沌期：回流、高潮、分歧，分歧延续后再做回流。
- 主升期：回流高潮后只接小分化，不一定有分歧延续，很快又回流；资金净值曲线持续往上。
- 退潮期：持续向下，修复后继续下跌；操作理念是管住手、恐惧为主，认知强才尝试政策或情绪反弹。

![三种周期的节奏对照](https://learn.luovista.com/trading/bibabu/member-hot-money-system/11/0450.jpg)

**视频时间点：07:30**

作者进一步说，小周期合起来就是大周期。比如龙头可以买首板、一进二，但那时通常还看不出谁是龙头；龙头一般在首次分歧后的转强，也就是二阶段，才更容易确认。确认龙头后，每天都有买点：核心弱转强、龙头弱转强、竞价、板上确认、第二天转强加速打板，甚至第二次分歧也是投机买点。

### 11:30 买在分歧是情绪周期的本质

作者强调：“买在分歧，是亘古不变的真理。”他反驳情绪周期无用论，认为那是对情绪周期理解不够深。情绪周期的核心不是画线，而是买在分歧。

他用不同投资流派类比：价值投资买低估值，本质是买在市场不认可它估值的时候；定投买回落，本质也是买在下跌分歧中；缠论讲背驰，背驰也是分歧；技术派讲支撑，一个票冲高后回落到支撑附近，本质也是分歧。很多投资理论拆开看，最终都离不开“买在分歧”。

![买在分歧的统一解释](https://learn.luovista.com/trading/bibabu/member-hot-money-system/11/0690.jpg)

**视频时间点：11:30**

### 13:20 买在分歧必须结合大周期：主升、混沌、退潮处理不同

作者提醒，买在分歧不是所有分歧都买。情绪周期涉及大周期和小周期：

- 混沌期买在分歧，卖在一致；高潮就是一致，连续分歧时才是较大的分歧。
- 主升期也买在分歧，但主升期高潮次日仍可接力，前提是审美提高，只接核心，不接后排。
- 退潮期反而不提倡随便买分歧，因为分歧后还可能继续分歧延续。

作者继续用白银有色说明：分歧第一天、第二天、第三天都不一定能买，要等有人去反核、市场出现跟随时，再考虑，而不是主观认为跌多了就买。

### 16:00 案例：文化传媒一类票在混沌期符合买在分歧

作者举了一个“文化传媒 / 传媒互联网”方向的票作为例子。它几次给出分歧，随后出现反包；高潮之后回调两天，再深水低吸，第二天又给反包。但他也提醒，后面再出现分歧时不一定能去，因为周末未必发酵起来，自己也没有去。

这个例子说明，混沌期很多操作是左侧预判，博弈下一天回流。买在支撑位置或分歧位置可以做，但不是所有分歧都必须做。

### 17:00 量化时代不是情绪周期失效，而是小周期循环加快

作者说，现在量化时代不是情绪周期没用了，而是小周期循环比以前快了。以前可能今天回流、明天分歧，再给半天或一天分歧，随后才是超跌买点；现在一天内就可能分歧、日内回流、日内再反杀。

![量化时代节奏加速](https://learn.luovista.com/trading/bibabu/member-hot-money-system/11/0960.jpg)

**视频时间点：16:00**

他举周五的盘面：早盘给分歧，日内也回流，但日内又接反杀，尾盘很多票跳水。买点逻辑本身没有错，但市场回流只持续半天，没有持续一整天，隔日就很难带出来。

因此量化时代更要有所为、有所不为，必须放弃一些小分歧节点。比如刚经历大亏钱效应后的分歧节点，和只是弱回流后的混沌分歧，质量完全不同。

### 19:20 什么样的分歧节点更值得博弈

作者用周二和周五对比。周二的全场情绪回流很强，因为前面已经经历两三天退潮，亏钱效应释放充分，赚钱效应在退潮后反弹，属于比较值得积极博弈的回流日。

周五的分歧延续则没有那么好：它不是主升后的分歧延续，只是混沌中的分歧；周三、周四的回流也是弱回流，并非市场整体性回流。这样的分歧延续去做就很难。

作者提醒，每天的题材节奏表要用来判断：哪些板块连续分歧，次日是否可以做回流；但这件事有成功也有失败，必须知道自己博弈的是分歧回流，失败就该卖、该止损。

![题材节奏表辅助判断](https://learn.luovista.com/trading/bibabu/member-hot-money-system/11/1220.jpg)

**视频时间点：20:20**

### 22:45 用计划成功率反推市场环境

作者说，可以通过他的计划成功率反推市场环境。他每天给三到四个备选计划，如果计划成功率特别高，毫无疑问是主升；如果计划成功率只有 15%，大概率是混沌；如果计划里都建议不要上车，大概率是退潮。

原因是他的自选都是核心票、板块核心，核心中的核心才会进入自选。观察这些核心票走势，就能知道当天情绪怎么样。

### 24:00 混沌期实操：在题材节奏表里找连续分歧板块

作者把方法落到盘面：如果知道现在是混沌期，混沌期买点在连续分歧，就到题材节奏表里找哪个板块昨天分歧、今天又分歧，明天能不能做回流。

他举例说，某些板块连续分歧后，次日弱分歧再接分歧，可能就可以做回流。天地在线是他做过的一个例子；如果选到神剑股份、红宝丽、百川股份、双良节能等类似分歧延续后的回流案例，也可能赚钱。但如果选错，比如选到航天发展，周五就会亏钱。

![连续分歧后的回流博弈](https://learn.luovista.com/trading/bibabu/member-hot-money-system/11/1380.jpg)

**视频时间点：23:00**

作者强调，有成功有失败，要把择股做好。择股要回看前面强势股择股那节课；本课只解决“什么时候买”的问题。

### 25:30 节奏错误：买在接力、分歧截面、主升空仓、退潮追涨

作者总结，混沌期买在分歧，尤其是买在分歧延续的低位，而不是买在接力。天地在线、红宝丽、华光环能、百川股份、双良节能等例子，都是他用来说明“分歧延续低吸 / 地板附近买入，次日反包或回流”的案例。

他也解释“分歧截面”：刚高潮完只给了弱分歧，你就急不可耐冲进去，随后还有分歧延续。你被连续分歧洗出去，结果后面又回流，这就是分歧截面。

![常见错误与正确节点](https://learn.luovista.com/trading/bibabu/member-hot-money-system/11/1660.jpg)

**视频时间点：27:40**

### 29:20 冰点不割肉，退潮不追涨

作者回顾自己的操作。北方铜业、同和药业等案例里，他曾经追进去被套；但周二是明确回流节点，冰点应该做的是低吸或做 T，而不是冰点割肉。

他以北方铜业为例，说自己在周二 9:50 附近低吸做 T，冲高后处理。这种冰点低吸做 T，才是冰点应该做的事：冰点试错、冰点买在卖核，而不是冰点割肉。

![北方铜业低吸做T示例](https://learn.luovista.com/trading/bibabu/member-hot-money-system/11/1810.jpg)

**视频时间点：30:10**

结尾作者把正确操作归纳为四条：

- 分歧不接面，不在分歧刚开始时追进去。
- 主升要重仓，不要空仓。
- 冰点不要割肉，而是尝试做反弹。
- 退潮不要追涨，退潮要休息。

他说，把这几个错误规避掉，账户至少能走平；如果一个错误都不犯，账户大概率能向上。若在一个周期里把这些错误全部犯完，回撤会很大。

## 本课术语表

| 术语 | 作者语境中的含义 |
| --- | --- |
| 大周期 | 混沌、主升、退潮三类市场环境，先决定操作大方向。 |
| 小周期 | 大周期内部的回流、高潮、分歧、分歧延续等短节奏，用于定位买卖点。 |
| 周期龙 | 被市场公认并能带动题材或情绪的龙头，作者用它判断是否进入主升。 |
| 混沌期 | 没有市场公认周期龙，多题材轮动，高潮后容易分歧。 |
| 主升期 | 有周期龙和主线，回流、分化、再回流不断上行，仓位比个股更重要。 |
| 退潮期 | 周期龙 A 杀或高位亏钱效应扩散，修复后容易继续分歧。 |
| 回流 | 资金重新流入某题材或情绪方向。 |
| 高潮 / 一致 | 题材强回流或一致性很高的位置，混沌期通常偏卖点。 |
| 分歧延续 | 分歧后继续释放亏钱效应，混沌期里常是博弈回流的前置条件。 |
| 分歧截面 | 刚开始分歧时急着买入，随后继续分歧，被迫割掉后又回流。 |
| 冰点 | 情绪极差、亏钱效应释放充分的位置，作者认为不应机械割肉。 |
