## 全片时间轴

### 00:00 本课定位：先会定义分歧和回流

作者开场说，这节课讲“板块的运行节奏”，核心是先学会定义什么是分歧、什么是回流。分歧可分为强分歧和弱分歧，回流也可分为强回流和弱回流；如果盘面看起来介于强弱之间，也可以加入“中分歧”“中修复”的概念。

![分歧回流定义框架](/images/trading/bibabu/member-hot-money-system/17/0005.jpg)

**视频时间点：00:05**

作者先给强分歧下定义：核心个股在深水或跌停，后排区域跌停，中军也出现负反馈，就叫强分歧。弱分歧则是核心个股震荡，最好是红盘震荡，后排分化或给机会。这里的关键不是某一只票涨跌，而是核心、后排和中军三类角色同时给出的反馈。

### 01:00 强回流和弱回流的定义

作者接着讲强修复，也就是强回流：核心个股不一定必须上板，但要能反包前一根阴线；后排跟随前排冲高，至少有一两个跟风涨停；中军也要表态。弱修复则是核心个股冲高，可能上板也可能不上板，但后排冲高回落，不能形成板块合力。

他强调，这些定义并不复杂，只要看得懂盘面，就能把当天板块定为强分歧、弱分歧、强回流或弱回流；中间状态则按中分歧或中修复处理。

### 02:38 算力案例：核心没有跌停时，不算强分歧

作者用算力题材举例。华胜天成和鸿博股份是当时更核心的个股。某一天鸿博股份高位放量、红盘震荡，电光科技在板上分歧放量，后排掉队，但作者不把它定义为强分歧，因为情绪核心没有炸板、没有给出极端负反馈，仍然在板上顶住。

![算力板块分歧案例](/images/trading/bibabu/member-hot-money-system/17/0165.jpg)

**视频时间点：02:45**

真正的大分歧，是后来鸿博股份低开去跌停，电光科技也明显弱化。作者说，这种核心个股去深水或跌停、板块核心给负反馈的状态，才符合强分歧。后面做预期时，要区分只是后排掉队，还是核心票和中军都在出问题。

### 04:10 化工案例：周期龙头跌停可定义为板块大分歧

化工案例里，作者提到中毅达被严监管延续影响，一字跌停；红宝丽也有负反馈，渝三峡 A 作为阶段情绪核心炸板回落。作者认为，虽然这个板块还穿越出红宝丽、红墙股份、渝三峡 A 等阶段票，但涨价逻辑真正的周期龙仍是中毅达。

所以当中毅达跌停时，就可以把化工定义为板块大分歧。作者在这里补充一个重要判断：不是只看当时最高位，也要看整轮题材最早、最核心的周期龙头是谁。

### 05:22 军工案例：分化不等于强分歧

军工某一天航天长峰开板放量，后排给机会，但最核心的成飞集成没有开板，甚至没有明显放量。作者把这种状态定义为分化，而不是强分歧。分化的含义是后排给机会、前排不给机会；它后面通常容易接真正的分歧。

作者用电力、机器人等案例说明，大部分板块强度没那么高时，常见节奏是“分化接分歧”。因此周末预期军工下一个交易日出现大分歧，是按大概率节奏做出的预期。

### 07:08 军工超预期：分歧后能强回流，才值得重视

作者问学员军工当天有没有分歧。他的回答是有分歧，因为早盘几乎所有票都给了机会，只是开盘后又转一致。能够在预期分歧后重新走强，说明这是一个特别强的板块。

他随后对比机器人：机器人曾经在两天分歧后接了强修复，作者当时把它定义为“首个分歧后接强回流的题材”，这种题材一定要重视。强回流后的题材可能继续高潮，甚至连续高潮；但连续高潮三天左右，后面就要警惕分歧。

![机器人强回流案例](/images/trading/bibabu/member-hot-money-system/17/0500.jpg)

**视频时间点：08:20**

### 09:20 机器人当下：中修复，还要看能否二次强回流

作者回到机器人当下盘面，认为当天更像低位中修复，不是强回流。万向钱潮承接了章盟主两个亿抛压，拓斯达成为低位 20 厘米首板，资金有高低切动作。中超控股是否算机器人核心并不清晰，因为它也有军工属性，早盘机器人冲高回落后它仍封住，不能简单说明机器人强。

作者认为，中信重工一类老龙头仍要持续关注。如果机器人想二次强回流，中信重工这类票可能绕不开；但低位新面孔更多是埋伏和看命，短线并不好做。

### 13:07 弱分歧与“三天小题材”

作者重新总结弱分歧：最核心的票不给机会，核心个股红盘震荡，后排无跌停或分化，这也可理解为常说的分化。正常预期下，分化后面容易接大分歧。

他讲“三天小题材”：第一天爆发，第二天分化，核心不给机会、后排部分跟上、大多数掉队，第三天大分歧。磁悬浮就是眼前案例。第一天嘉力图上板，后排山东章鼓、百达精工、新美星、磁谷科技等跟随发酵；第二天嘉力图一字，其他后排水下开或冲高回落；第三天多数票走弱。作者说，如果手上不是绝对核心，第二天就要先出掉。

### 16:10 纺织与核电案例：爆发后直接分歧，也可能题材结束

作者把纺织和核电放在一起讲。核电当时启动后，第二天直接大分歧，板块基本结束。纺织受中美缓和消息刺激，华纺股份快速上板，迎丰股份等后排跟随；作者自己当天半路了一个首板迎丰股份，第二天躺了一字，但板块大分歧时他就出了。

他提醒，迎丰股份当天虽然扛住，但次日是否有溢价很难说，可能像嘉力图一样补跌。对于小题材，不能因为某一天核心扛住就默认题材升级。

### 18:08 强回流的核心检验：反包阴线，中军也表态

作者再次复述强回流标准：核心个股上板或至少反包前面阴线，后排跟随冲高，至少有一两个跟风涨停，中军也要表态。

机器人强回流案例中，中信重工竞价弱转强，后排跟随，最强分支是 PEEK 等；当日反包前一天断板阴线。军工案例中，中航成飞作为中军，前一日爆量 80 亿后，次日还能反包阴线，即便有呼家楼大资金推动，也不能否认这就是强回流。作者说，中军就是用来锚定板块节奏的。

![强回流与中军表态](/images/trading/bibabu/member-hot-money-system/17/1110.jpg)

**视频时间点：18:30**

### 20:12 周五盘中强回流不算收盘强回流

有学员问周五是否算强回流。作者区分盘中和收盘：周五先是竞价一致转分歧，成飞集成给机会；随后航天长峰回封，盘中转一致；后来航天长峰又炸板，收盘时后排和中军都有负反馈。所以从收盘看，周五不算真正强回流。

而下一个交易日，军工在竞价走弱情况下还能强反包，很多持有者甚至没想到还能赚钱走，这就是超预期。作者强调要理解这种持筹者的心理状态：自己拿着都没想到能赚钱，说明盘面比预期强。

### 21:45 弱修复是卖点，但要结合题材预期

作者用机器人当天走势讲弱修复：早盘核心或双题材个股冲高，但后排、中信重工等跟随冲高后走弱，这就是弱修复。弱修复一般是卖点。

但他补充，若自己长期看好一个题材，比如华为链后续还有华为手表、电脑等消息刺激，资金介入也比较深，仓位又不重，那么弱分歧或弱修复时也可以多拿一拿。这里不是因为弱修复变强，而是题材后续预期改变了持仓处理。

### 24:13 总结：强弱看核心、后排和中军是否给机会

作者做一个简短总结：强弱分歧可以理解为看个股给不给机会，强弱回流则看核心能不能反包、后排能不能跟、板块能不能形成合力。强分歧是核心给深水或跌停机会，后排大面积负反馈，中军也弱；弱分歧是核心不给机会或红盘震荡，后排分化。

回流也是一样，强回流要求核心、中军和后排共振；弱回流则只是核心或个别票冲高，后排冲高回落。介于两者之间，就定义为中修复。

### 26:00 板块节奏的连续性：爆发、高潮、分歧、回流

作者讲，正常板块运行不是每天都能买。爆发和高潮之后容易接分歧，分歧之后才有回流预期；回流可能是弱回流，也可能是强回流。如果分歧后不能回流，板块就可能结束；如果分歧后强回流，则要重视潜在主线。

作者提醒，不能在一致高潮时再去接后排。买点通常在分歧或回流开始，而不是一致之后。

**视频时间点：25:00**

### 32:00 主流题材给更多耐心，但仍要分清前排和后排

作者说，真正主线题材和普通题材不同。普通高潮次日接分歧，后排通常是卖点，前排才可能是买点；但真正主线分歧时，后排中也可能出现买点，因为主线会出现卡位，今天的前排不一定是明天的前排。

军工当天就是例子。竞价分歧时，天箭科技、航天长峰、立航科技等后排或换手票也可能成为买点，因为主线题材给资金更多容错。但这不代表任何后排都能买，前提是板块确实具备主线强度。

### 47:00 连续分歧后的尾盘买核心：博弈次日修复

作者讲一个手法：连续分歧的逻辑题材，第三天往往有修复预期，修复力度不确定，可能强也可能弱，但至少有弱修复的概率。尾盘去拿板块核心，第二天只要冲红，就有套利空间。

![连续分歧尾盘买核心](/images/trading/bibabu/member-hot-money-system/17/3030.jpg)

**视频时间点：50:30**

他用华为链和青岛金王举例：连续分歧两天后，尾盘拿核心，第二天若弱修复红开，就能赚钱走。这个手法在混沌期尤其常用，因为混沌期很难连续三天一直向下杀，除非进入退潮期。作者强调，要买板块核心，不要买杂毛。

### 51:50 尾盘抢筹：对先手是利好，对次日接力是利空

作者解释为什么会出现尾盘抢筹。很多资金都知道连续分歧后次日有修复预期，于是都在尾盘拿先手，盘面就形成尾盘抢筹。

![尾盘抢筹的供需解释](/images/trading/bibabu/member-hot-money-system/17/3300.jpg)

**视频时间点：55:00**

对已有先手的人来说，尾盘抢筹是利好，因为当天就有盈利；但对第二天想来做修复的资金来说是利空，因为需求被提前消耗，次日接力需求减少，而今天拿先手的人明天会成为供给。作者用供需解释：股票上涨需要需求大于供给，尾盘抢筹会让次日需求减少、供给增加，原本略微大于供给的需求可能变成小于供给。

作者又用通俗说法总结：尾盘抢多少，第二天正常预期就低开多少。比如原本低开 1 个点、冲红有套利空间；尾盘抢到正 2，次日可能低开 3，哪怕上冲 1.5 个点，也仍然没有溢价。

### 59:00 尾盘抢筹不绝对，离别钩要小心

作者补充，尾盘抢筹有好有坏，逻辑特别强的板块也可能强更强。如果中军略微抢筹后又被砸下来，次日仍可回归正常修复预期。但如果连板个股出现“离别钩”，就要小心。

![离别钩案例](/images/trading/bibabu/member-hot-money-system/17/3620.jpg)

**视频时间点：60:20**

作者定义离别钩：连板个股尾盘无量快速拉升，不带板块，可能过不了均线，也可能刚过均线就被砸下来。越高位的连板票，离别钩越夸张，甚至能从水下拉到六七个点，让人以为要地天，结果碰到均线就回落。若已有先手，这种地方应该卖掉。

### 63:30 盘中如何确认不是轮动：看板块整体和核心个股

有学员问军工竞价低开后，如何判断它不是普通轮动。作者说，把板块涨幅榜、涨速榜打开，再叠加核心个股分时。只要板块整体往上动，就可以买，但前提是买核心、买辨识度。

他回看自己的复盘计划：板块整体不及预期，回流就直接干核心；再次强化回流，也直接干核心。作者说自己很少写“直接干”这类话，这来自对板块预期和核心票强度的判断。

### 65:00 当日复盘：机器人、化工、算力和华为链

作者随后做当日盘面复盘。机器人看能否二次回流，中超控股更偏双题材，既有军工也有机器人；中信重工作为老龙头趋势没坏，需要继续关注，但作者早盘已卖出。

春光科技作为最高板，作者认为次日正常预期是断板，因为当天更多是被机器人板块反推上去。机器人最大问题是趋势向上，但连板持续性不强。

化工方面，中毅达释放流动性开板，红墙股份反包二板，渝三峡 A 地天后炸板，红宝丽企稳，因此化工当天也算中修复。

算力和华为链属于连续分歧后止跌，次日若接回流就能看，不接回流就走完。作者认为老题材里机器人和化工已修复，唯一没修复的是算力，所以轮动资金尾盘拿算力、华为链先手可以理解。

### 73:00 军工高潮后的处理：短线先兑现，二次介入等分歧

作者说军工当天高潮后，次日若高开，自己倾向先砸一半。短线不是波段，没预期的板块先跑再说。他认为成飞集成作为“力挽狂澜”个股，分歧日主动带动回流，次日有一字加速预期；但中航成飞这种容量中军，因为张盟主持货太多，次日不适合再去接力。

作者提出买点位置：买票要买在分歧，左侧博弈回流；或者买在回流刚确认，右侧跟随。不能买在一致。强更强之后再强更强，就是一致更加一致，一般不能接。

![左侧分歧和右侧回流买点](/images/trading/bibabu/member-hot-money-system/17/4520.jpg)

**视频时间点：75:20**

### 81:20 中巴铁路和轨交设备：新题材发酵不充分则次日按预期处理

作者提到当天唯一新题材是中巴铁路，包括中巴铁路和中巴卫星，背景是巴西访问前关于基建、航空航天合作的预期。他半路了神州高铁，因为轨交设备是除军工外涨幅靠前的概念，也有二次回流动作。

但板块最终没有发酵成功，神州高铁也没封住。作者说次日就按正常预期处理，若分化走弱就走。这段说明，新题材即使有消息和板块异动，也要看能否真正发酵成功。

### 84:25 大盘和周期：混沌期里看潜在主线

作者复盘大盘预期：高开后是直接兑现，还是兑现后接回流，需要晚上重点思考。对短线来说，非核心个股先兑现一部分。

他认为大周期仍是混沌期，不是主升。主升应由连板个股打开高度，但当时连板个股如春光科技更多是被板块反推，个体强度不够。军工具备潜在主线预期，甚至也可以直接按主线做，但持续性还需要再观察。

### 89:30 次日计划：开仓延后，等兑现后的回流

作者说，次日若短线情绪和大盘共振回暖后竞价再高开，就构成强更强，兑现为主，开仓延后。即便要买，也要晚一点，不能竞价直接干进去。

计划上，先持股观望或先考虑兑现，若盘中回流再跟随买票。能看的方向包括军工、化工核心红墙股份反包、算力核心华胜天成或鸿博股份低吸、机器人中超控股或低位首板、消费修复等。但他也指出，确定性很高的方向并不多。

### 94:30 答疑：万向钱潮、ST、没空看盘和纺织逻辑

作者答疑中讲到万向钱潮为什么被顶一字：盘前发酵了“万向手”消息，冠盛股份和万向钱潮都被提到，冠盛股份偏后排，万向钱潮是前排。万向钱潮分歧，后排分歧就很正常。

他也提醒 ST 不要做，不是赚多赚少的问题，而是账户和交易限制风险。没空看盘的人，重点看 9:15 到 9:45。纺织走差的原因是逻辑不够硬、没人持续吹，属于三天小题材，第二天走弱后后排先卖。

### 97:30 行情差时怎么做：空仓或只做核心修复

有学员问行情很差怎么做。作者回答很简单：要么空仓，要么清仓。他也承认自己做得不够好，之前几天红多绿少但账户仍小绿。

最近行情难做的原因是高位监管、低位轮动太快。要么打首板，第二天走弱就走；要么做高位绝对核心，博弈它的回流修复，其他都不好做。

### 99:00 新题材、强题材和低吸轮动

作者引用一个思路：有新做新；没有日内新题材，就做老题材里比较强的方向；再不行，就是低吸轮动，看命。

![新题材与低吸轮动](/images/trading/bibabu/member-hot-money-system/17/5010.jpg)

**视频时间点：83:30**

上周五有纺织这个新题材，他做迎丰股份；当天如果没有新题材，就可以看盘中走强的军工，或者低吸老题材里的机器人、化工、算力。低吸到机器人和化工就是命好，低吸到算力就暂时不赚钱。

作者说，盘中发现板块发酵，通常看涨速榜，再点进板块看 K 线和筹码，如果结合前面课程讲的低位主力拿货图形好看，就可能直接半路。

### 103:30 有新冲新、没新冲强，连续分歧低吸核心

作者继续强调三条执行线：有新题材就先冲新题材；没有新题材就冲当天走强的板块；低吸轮动则是连续分歧的板块，低吸它的核心个股，博弈后续回流。

他以神州高铁和宗申动力举例。神州高铁是中巴铁路/轨交设备方向的半路案例；宗申动力则不能只理解为低空三傻之一，它还有军工、机器人、低空等多重逻辑，且缩量回调、主力资金没有明显流出，所以可按趋势观察。

### 107:53 尾声补充：指数走好后看量能和潜在主线

视频尾部还有一段补充讨论。作者说，既然指数趋势走出来了，就先乐观，但短期必须看到市场放量，并按正常市场节奏去走。若指数走好，后续要看哪些板块容易成为主线，至少要有主升预期。

![尾声量能与主线讨论](/images/trading/bibabu/member-hot-money-system/17/5730.jpg)

**视频时间点：95:30**

作者认为，当天盘面是近段时间难得比较符合短线模型的现象：周五静态复盘，加上当天右侧动态反馈，走出了符合短线的一种节奏。次日重点看量能，只要量能能放到约 1.15 万亿，分歧时可以大胆干；军工更适合短线选手，机器人更适合耐心型选手。

## 本课术语表

| 术语 | 作者语境中的含义 |
| --- | --- |
| 强分歧 | 核心个股深水或跌停，后排区域跌停，中军也出现负反馈。 |
| 弱分歧 | 核心个股红盘或震荡，后排分化，通常也可称为分化。 |
| 中分歧 | 介于强弱分歧之间的状态，由盘面强度自行分层。 |
| 强回流 / 强修复 | 核心个股反包前阴线，后排跟随冲高并有涨停，中军表态。 |
| 弱回流 / 弱修复 | 核心个股冲高，但后排冲高回落，板块合力不足。 |
| 三天小题材 | 第一天爆发，第二天分化，第三天大分歧的小级别题材节奏。 |
| 尾盘抢筹 | 资金提前博弈次日修复，在尾盘集中拿先手。 |
| 离别钩 | 连板个股尾盘无量快速拉升，不带板块，触均线或过均线后迅速回落。 |
| 左侧买分歧 | 在板块分歧时买核心，博弈后续回流。 |
| 右侧买回流 | 回流信号确认后跟随买入，如核心上板、后排跟随。 |
