## 课程信息

- 课程编号：第 1 课。
- 课程定位：整套游资交易系统的总入口和基础认知课。
- 主要任务：区分投机与投资，建立短线盈利的三个基础支柱，并说明工具、资金和心态准备。
- 本课不解决：具体买点、选股规则、情绪周期划分和战法执行。

## 本课在交易系统中的位置

![游资交易系统课程大纲](/images/trading/bibabu/member-hot-money-system/01/0012.jpg)

**视频时间点：00:12**

这节课不是具体战法教学，而是在回答“后面的知识应该装进什么框架”。作者给出的课程顺序大致是：

```text
短线交易基础
→ 量价与看盘
→ 复盘和工具
→ 卖点、情绪周期
→ 择时、择股、交易心法
→ 具体战法
```

作者明确把战法放在后面，并反复强调：**战法是次要的，择时和情绪周期理解更重要。**

这意味着后续课程不能被理解成一组彼此独立的“技巧”。作者设想的系统是先判断环境和资金，再选择模式；具体战法只是最后的执行工具。

## 本课核心结论

1. 短线交易属于投机，主要赚取短期价格波动差价，不能与长期投资逻辑混用。
2. 短线盈利依赖三个基础：市场情绪、资金流动、胜率与赔率。
3. 资金倾向从亏钱效应方向流向赚钱效应方向。
4. 单笔交易是否盈利不重要，模式的长期正期望才重要。
5. 新手必须用小仓位降低试错成本，并接受概率性亏损。
6. 看盘软件和 L2 是作者认为需要提前准备的基础工具，但工具不能替代交易逻辑。

## 一、核心概念与作者定义

### 投机和投资不能混成一套逻辑

![投机、投资与准备工作](/images/trading/bibabu/member-hot-money-system/01/0095.jpg)

**视频时间点：01:35**

作者用持有周期和盈利来源区分两者：

| 维度 | 投机 | 投资 |
| --- | --- | --- |
| 盈利来源 | 短期价格波动差价 | 公司基本面长期增长 |
| 典型周期 | 一至数日或更短 | 通常按年计算 |
| 研究重点 | 情绪、资金、价格、交易模式 | 公司价值和经营增长 |
| 代表人物 | 杰西·利弗莫尔 | 巴菲特 |

本课的重点不是争论哪一种更高级，而是要求交易者先确认自己究竟在做什么。做短线却在被套后临时讲长期价值，或者拿长期投资方法解释日内波动，本质上都是逻辑错位。

作者允许同时学习投机和投资，但建议基础薄弱者先专攻一个方向。

### 短线盈利的三个根源

作者将短线盈利归纳为：

- **市场情绪**：买在情绪低点，卖在情绪高点。
- **资金流动**：资金从阻力大的亏钱效应方向，流向阻力小的赚钱效应方向。
- **概率游戏**：把胜率和赔率结合起来，追求模式的正期望。

这三个支柱分别回答：

```text
现在是否适合出手？
资金更可能去哪里？
长期重复这笔交易是否划算？
```

## 二、作者的完整讲解路径

作者先解释投机与投资，避免学习对象错位；然后讲短线盈利的情绪、资金和概率基础；最后补充看盘工具、资金管理和心态建设。

整个路径可以压缩为：

```text
确认交易类型
→ 理解市场和资金
→ 用胜率、赔率评价模式
→ 准备观察工具
→ 控制初始风险
→ 接受单笔随机性
→ 重复正期望模式
```

这一顺序很有意味：工具和战法都没有被放在最前面。作者认为，看得见盘口不等于能理解资金，学会一个战法也不等于知道何时使用。

## 三、决策流程与操作步骤

### 第一步：先判断自己在做投机还是投资

短线交易应使用投机框架，关注短期价格和资金行为。如果更认同长期基本面方法，就应明确转向投资，而不是交易中途切换解释体系。

### 第二步：先看情绪位置

作者本课只给出总原则：

> 买在情绪低点，卖在情绪高点。

具体识别方法留给后续情绪周期课程。因此，目前不能把这句话直接机械化为“跌了就买、涨了就卖”；情绪位置需要后续课程补充可观察条件。

### 第三步：观察资金阻力

作者把亏钱效应理解为资金流动的阻力。一个方向持续亏钱，资金继续进入的阻力就大；一个方向形成赚钱效应，资金更容易跟随。

因此，资金流向判断不是只看某笔“大单净流入”，而是要判断哪个方向真正形成了可持续赚钱效应。

### 第四步：计算模式是否值得重复

![胜率与盈亏比表](/images/trading/bibabu/member-hot-money-system/01/0335.jpg)

**视频时间点：05:35**

作者用盈亏比说明：胜率不是越高越好，赔率也不能单独看。两者必须组合。

假设每次亏损为 1：

| 盈亏比 | 理论盈亏平衡胜率 | 本课含义 |
| --- | ---: | --- |
| 1:1 | 50% | 盈亏相同，需要至少一半交易盈利 |
| 1:1.5 | 40% | 视频举例中，约 40% 胜率可接近平衡 |
| 1:2 | 33.3% | 作者按展示表格近似表述为约 35% |
| 1:5 | 16.7% | 少量成功交易可以覆盖多次小亏损 |

计算关系为：

```text
盈亏平衡胜率 = 单次亏损 ÷（单次盈利 + 单次亏损）
```

这里的关键不是背数字，而是建立模式统计意识：如果不知道一套模式长期的胜率、平均盈利和平均亏损，就无法判断它究竟是有效交易还是偶然赚钱。

### 第五步：小仓位验证

新手不要一开始投入全部本金。作者强调“盈亏同源”：能够快速复利的方式，也可能快速造成巨额亏损。因此应主动缩小仓位和资金体量，用较低成本完成模式学习和试错。

### 第六步：评价一组交易，而不是纠结一笔交易

单笔交易可能遭遇减持、突发事件或其他黑天鹅。作者要求接受这种随机性，把注意力放在模式的长期有效性上。

## 四、案例拆解

### 案例一：1:2 盈亏比

- 预期盈利：10%。
- 可接受亏损：5%。
- 盈亏比：1:2。
- 作者结论：胜率约 35% 能接近盈亏平衡，达到 40% 后可能形成正收益。
- 学习重点：不必追求每笔都对，关键是盈利交易能否覆盖亏损交易。

### 案例二：1:5 盈亏比

- 预期盈利：5%。
- 可接受亏损：1%。
- 盈亏比：1:5。
- 理论盈亏平衡胜率：约 16.7%。
- 学习重点：高赔率模式可以容忍较低胜率，但前提是能够严格把亏损限制在计划范围内。

### 案例三：突发黑天鹅

作者用减持、离婚和“扇贝跑了”等事件说明，单笔交易可能遭遇模式之外的突发风险。这类亏损不能通过复盘强行解释成“本来就应该预见”，真正要验证的是同类交易样本的整体期望。

## 五、新手误区与交易纪律

### 常见误区

- 做短线投机，却用长期价值投资解释被套。
- 最想先学具体战法，忽略择时和情绪周期。
- 只追求高胜率，不计算单次盈利和亏损的比例。
- 看到大单净额就直接等同于真实资金方向。
- 刚开始学习就投入全部本金。
- 因一笔亏损否定整套模式，或因一笔盈利认定模式有效。

### 本课纪律

- 先确定交易类型，再选择知识体系。
- 不在情绪高点机械追涨杀跌。
- 用小仓位和小资金降低试错成本。
- 接受模式内的正常亏损和无法预测的单笔黑天鹅。
- 用一组交易的胜率和赔率评价模式。

## 六、适用条件与失效边界

### 适用条件

- 用于建立短线交易的基础评价框架。
- 用于比较不同模式是否具备长期正期望。
- 用于新手确定学习顺序和初始风险预算。

### 不能直接套用的地方

- “情绪低点”尚未给出识别标准，不能简单等同于指数下跌或个股回调。
- 预期盈利和止损必须能够真实执行，否则名义盈亏比没有意义。
- 胜率需要来自足够数量、定义一致的历史交易，不能凭主观感觉估计。
- 手续费、滑点、涨跌停和无法成交等问题会提高实际盈亏平衡胜率。
- L2 展示的是交易数据，不会自动给出买卖结论。

## 七、可执行检查清单

每次计划一笔短线交易前，至少回答：

- [ ] 我是在做短线投机，还是长期投资？
- [ ] 当前情绪位置是什么？判断依据是什么？
- [ ] 当前赚钱效应和亏钱效应分别在哪里？
- [ ] 资金为什么可能从原方向流向目标方向？
- [ ] 计划亏损多少，计划盈利多少？
- [ ] 这套模式的历史胜率和平均盈亏比是多少？
- [ ] 当前仓位是否允许连续多次试错？
- [ ] 如果出现模式外黑天鹅，最大损失是否可承受？
- [ ] 我是在执行已定义的模式，还是因为临盘情绪改变计划？
