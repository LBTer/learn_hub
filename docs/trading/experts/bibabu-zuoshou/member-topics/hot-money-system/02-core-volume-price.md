## 课程信息

- 课程编号：第 2 课。
- 课程定位：把第一课的“资金流动”转化为可从 K 线和成交量观察的行为。
- 主要任务：识别短线见顶、短期止跌、健康拉升和主力吸筹。
- 核心边界：顶部信号可偏重风险处置；底部与吸筹信号必须组合使用。

## 本课在交易系统中的位置

```text
市场与资金基础
→ 读取价格和成交量
→ 判断顶部风险 / 短期止跌 / 健康趋势 / 主力吸筹
→ 再结合支撑压力、情绪周期和催化执行
```

本课没有给出独立战法，而是在建立后续选股、择时和卖点课程共同调用的“量价语言”。

## 本课核心结论

1. 高位爆量长上影、爆量十字星和封死跌停，是短线强势股较重要的退出信号。
2. 有先手时先控制风险；次日真正修复或反包，再买回来，比硬赌反包更符合作者纪律。
3. 短期止跌通常先经历极致缩量和振幅收敛，再由放量阳包阴或连续小阳线确认。
4. 单个见底信号可靠性有限，位置、支撑、趋势、市场周期和多个信号叠加更重要。
5. 健康拉升常表现为量价齐升；趋势未被明确顶部信号或结构破坏否定前，不轻易看空。
6. 主力吸筹的典型痕迹是“放量阳线 + 缩量阴线”和“连续小碎步阳线”。
7. 跌破趋势线不能只凭“震仓”叙事硬扛，应先退出，再观察是否重新企稳。

## 一、核心概念与作者定义

### 放量滞涨

![三联锻造爆量长上影](/images/trading/bibabu/member-hot-money-system/02/0092.jpg)

**视频时间点：01:32**

放量滞涨不是单纯“成交量大”，而是成交量显著放大后，价格无法继续有效上涨。作者进一步拆成爆量长上影、爆量十字星和高位封死跌停。

### 短期止跌

![课程量价关系体系图](/images/trading/bibabu/member-hot-money-system/02/0689.jpg)

**视频时间点：11:29**

作者讲的底部主要是短期回调底部，服务于超跌反弹。它通常包含两个阶段：

```text
下跌动能衰减：极致缩量 + 振幅变小
→ 买盘确认：放量阳包阴 / 连续小阳线
```

### 主力吸筹

吸筹不是看到一根放量阳线便确认主力，而是观察一段时间内是否反复出现上涨放量、回调缩量，或者连续小阳线缓慢推升。

## 二、作者的完整讲解路径

作者先从风险最高的顶部信号讲起，再讲短期止跌和超跌反弹，随后讲健康拉升，最后讲主力建仓。

这个顺序实际对应四个交易问题：

```text
什么时候先卖？
什么时候可能止跌？
趋势是否仍健康？
启动之前资金是否已经拿货？
```

## 三、决策流程与操作步骤

### 顶部风险处理

1. 确认是否位于高位或强势趋势后段。
2. 寻找爆量长上影、爆量十字星或首次封死跌停。
3. 有先手时先退出，避免把风险控制变成反包赌博。
4. 次日若修复，再重新评价；若同时跌破五日线或趋势线，顶部信号更强。

### 短期止跌判断

1. 先找极致缩量和振幅收敛。
2. 再找放量阳包阴或连续两根小阳线。
3. 检查是否靠近前期支撑、W 底或黄金分割位置。
4. 结合市场是否进入新周期，而不是脱离环境机械抄底。

### 主力建仓判断

1. 检查一段时间内阳线是否放量、阴线是否缩量。
2. 判断结构是箱体震荡还是小斜率向上。
3. 观察后期是否进一步缩量，意味着浮筹减少。
4. 同题材候选股中，优先研究有明显建仓痕迹者。
5. 若趋势线被放量阴线跌破，先处理风险，再验证是否为震仓。

## 五、新手误区与交易纪律

- 把任何一根放量 K 都叫主力吸筹。
- 高位见顶信号出现后，仍只想着少数反包案例。
- 看到极致缩量便立即抄底，没有等阳包阴或连续小阳线确认。
- 把短期反弹底误认为长期趋势底。
- 趋势线跌破后，用“主力震仓”给亏损找理由。
- 只看图形，不看所处位置、市场周期和资金结构。

本课最鲜明的纪律是：**顶部先处理风险，底部必须等待组合确认。**

## 六、适用条件与失效边界

### 适用条件

- 短线强势股和趋势股的风险管理。
- 超跌反弹的初步筛选。
- 同题材候选股中比较筹码和资金准备程度。

### 失效边界

- 量化或其他新资金可能接力原主力出货后的股票，令标准顶部信号失效。
- 流动性差的小盘股容易出现虚假放量、缩量。
- 消息刺激、停牌重组和内幕信息会改变正常量价解释。
- “爆量”“极致缩量”“微微放量”尚无统一量化阈值。
- 单个阳包阴或两根小阳线只能说明短期止跌概率提升，不保证反转空间。

## 七、可执行检查清单

- [ ] 当前处于高位、回调底部、健康趋势，还是建仓阶段？
- [ ] 成交量相对近期均量究竟放大了多少？
- [ ] 高位是否出现长上影、十字星或首次封死跌停？
- [ ] 顶部信号是否叠加五日线或趋势线破坏？
- [ ] 底部是否先经历极致缩量和振幅收敛？
- [ ] 是否出现放量阳包阴或连续两根小阳线？
- [ ] 是否靠近可解释的支撑位置？
- [ ] 当前市场环境是否支持超跌反弹？
- [ ] 上涨是否反复放量、回调是否反复缩量？
- [ ] 如果判断“震仓”，重新企稳的客观证据是什么？
