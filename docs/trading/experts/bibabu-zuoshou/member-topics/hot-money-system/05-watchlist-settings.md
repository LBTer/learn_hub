## 一、本课定位与学习目标

- 系统位置：承接第 4 课的盘后复盘流程，把复盘得到的板块、核心票、计划票和预期放进盘中看盘界面。
- 前置知识：需要理解情绪、量价、支撑压力、板块角色和复盘票池。
- 本课解决：盘中看什么、怎么看资金流动，以及同花顺界面如何服务于短线看盘。
- 学完能够：用竞价封单、涨幅 / 涨速榜、短线精灵、涨停分析、分时叠加和盘口界面，识别板块回流、分化、对流和联动。
- 本课不解决：具体买点、卖点、仓位、盘口战法细节和不同模式的完整选股规则。

本课最重要的转变，是从“盯持仓”转为“看资金流动”。作者认为，只看持仓股属于无效看盘；真正的盘中任务，是在处理完持仓计划后，持续观察资金正在进攻哪里、哪里被吸血、哪些板块反馈一致或分化。

## 二、核心规则与知识结构

### 核心规则

1. 看盘的目标是洞察资金流向，不是机械盯价格。
2. 竞价封单看一字强弱：封单加大通常代表方向可能加强，封单递减则要警惕板块分歧。
3. 涨幅榜和涨速榜分工不同：涨幅榜看当天主攻方向，涨速榜看当下回流方向。
4. 板块强度要看核心与后排是否一致，前排回流不等于板块高潮。
5. 一字板叠加快速自然涨停，说明该方向当天发酵概率较大。
6. 反复出现的“一个方向涨、另一个方向跌”，才更可能构成资金对流或吸血关系。

### 知识结构

```text

盘前 / 复盘
→ 明确持仓计划、板块核心、计划个股

→ 盘中：先处理持仓，再看资金流动

→ 个股：短线精灵 + 个股涨跌速 + 分时叠加 + 盘口
→ 判断：进攻、回流、分化、对流、吸血、联动
```

## 三、决策流程与使用边界

### 竞价阶段：用封单和一字方向建立先验

- 观察信号：一字板方向、9:15 到 9:25 封单变化、实际封单额、竞价金额、快速自然涨停。
- 判断条件：封单相较前一日明显加大，代表个股和方向可能加强；封单持续递减，代表方向可能进入分歧。
- 对应动作：9:25 后先弄清一字板属于什么板块、为什么涨停，再观察开盘后的快速自然涨停是否与一字方向呼应。
- 常见误用：只看是否一字，不看封单变化；或看到一字就默认板块全天强。
- 失效边界：一字强不代表后排一定跟随，仍要看自然涨停、后排反馈和盘中承接。

### 盘中资金流动：涨幅看主攻，涨速看当下

- 观察信号：板块涨幅榜、板块涨速榜、个股涨跌速榜、短线精灵。
- 判断条件：板块涨速榜出现某方向，同时个股涨速榜和短线精灵也出现该方向核心票，说明当下更可能是该方向回流。
- 对应动作：把板块和个股信号交叉验证，再判断资金是在进攻、回流还是只拉单票。
- 常见误用：把当天涨幅高的方向等同于当下正在进攻；或只看单个异动个股，忽略板块是否跟随。
- 失效边界：1 分钟涨速太敏感、噪音大；作者建议个股涨跌速榜用 3 分钟，避免无效信息过多。

### 板块反馈：核心票和后排必须一起看

- 观察信号：核心票是否上冲，后排是否跟随，板块内是否全红，辨识度票是否仍在水下。
- 判断条件：核心强、后排也普遍红盘，才更接近一致或高潮；核心强但后排很多水下，多数只是分化或前排抱团。
- 对应动作：对热点板块保留核心票观察区，盘中持续看前排和后排状态。
- 常见误用：前排拉升就判断板块高潮。
- 失效边界：板块刚启动时，角色和后排反馈可能还不稳定，需要后续自然涨停和短线精灵继续确认。

### 今日涨停：一字加快速自然涨停是发酵信号

![今日涨停按首次涨停时间排序](https://learn.luovista.com/trading/bibabu/member-hot-money-system/05/0925.jpg)

**视频时间点：15:25**

- 观察信号：9:25 后的一字板，开盘后的自然涨停，首次涨停时间排序。
- 判断条件：一个板块既有一字板，又出现快速自然涨停，当天发酵概率较大。
- 对应动作：用短线侠或开盘啦看一字，用同花顺涨停分析按首次涨停时间看自然涨停。
- 常见误用：只看涨停数量，不看涨停出现的时间顺序和一字 / 自然涨停组合。
- 失效边界：如果封单明显减少，或者快速涨停后后排分化严重，要降低板块预期。

### 分时叠加：识别风格 PK 和资金对流

- 观察信号：两个或多个核心票的分时同步或相反变化、涨速榜排名、大单漏单、炸板时间。
- 判断条件：一个方向拉升时，另一个方向反复走弱，且出现多次，才更可能是对流或吸血关系。
- 对应动作：把核心票分时叠加，观察拉升、炸板、漏单和回封是否在同一时间段发生。
- 常见误用：只看到一次相反走势就断定两者对流。
- 失效边界：第一次出现可能只是偶然，必须经过两三次以上重复确认。

### 界面设置：信息要服务于盘中判断

![自选股与盘口界面设置](https://learn.luovista.com/trading/bibabu/member-hot-money-system/05/1525.jpg)

**视频时间点：25:25**

- 观察信号：连板区、炸板 / 断板区、板块统计、自选核心票、计划票、竞价监控、L2 委托明细、盘口、分时、日 K、短线精灵、消息快讯。
- 判断条件：界面不是越多越好，而是要让持仓、计划、板块、资金流动和盘口细节都能快速切换或同时观察。
- 对应动作：条件允许时双屏：一个屏看自选股和盘口，一个屏看板块统计；屏幕有限时来回切换也可以。
- 常见误用：设置很多栏目但没有明确用途，或只放持仓导致看不到全市场流动。
- 失效边界：盘口细节和 L2 需要后续专门训练，本课只讲界面位置和观察入口。

## 四、典型案例与对照

| 案例 | 教学作用 | 类型 | 关键结论 |
| --- | --- | --- | --- |
| 摩恩电气 | 展示竞价封单加强如何对应板块加强 | 主案例 | 封单从弱到强，电力 / 变压器方向预期提高 |
| 海陆重工 | 展示封单递减如何预示板块分歧 | 对照案例 | 封单越来越少，核电方向后排容易冲高回落 |
| 方正电机 + 快速自然涨停 | 展示一字叠加自然涨停的发酵信号 | 规则案例 | 一字板方向若出现快速自然涨停，当天发酵概率较大 |
| 漳州发展 vs 德明利 | 展示投机与趋势的风格 PK | 主案例 | 漳州发展带动海峡回流时，德明利漏单，资金从趋势向投机切换 |
| 长城军工 + 北方长龙 | 展示个股联动关系 | 补充案例 | 两只票反复同时出现在涨速榜，说明市场已形成联动记忆 |

### 主案例：摩恩电气的封单加强

- 案例背景：摩恩电气作为电力 / 变压器方向的一字核心。
- 作者拆解：前一日 9:25 只剩约 2.7 亿元封单，次日 9:25 剩约 20.5 亿元，属于明显加强。
- 对应规则：竞价封单不仅看开出来多少，更看 9:25 结束后还剩多少，以及相较前一日是否加强。
- 关键结论：封单加强会提升板块加强概率；周四电力方向后续走出高潮，和这个竞价信号相互印证。

### 对照案例：海陆重工封单递减后的核电分化

- 对照关系：海陆重工前期封单尚能体现认可，后续封单逐步减少。
- 容易误判之处：只看它最终封住，就忽略封单和后排强度下降。
- 作者结论：封单递减时，要想到板块可能大分歧。当天核电只有海陆重工、兰石重装等少数票上板，其他多冲高回落。

### 主案例：漳州发展与德明利的风格之争

- 案例背景：周五下午海峡方向回流，趋势方向德明利、香农芯创冲高回落。
- 作者拆解：漳州发展约 13:43 开始拉升，13:46 附近拉到高点；同一时间德明利出现大单漏单。投机方向看到趋势方向不回封，就抢这个节点继续上攻。
- 对应规则：分时叠加可以还原资金对流，涨速榜和盘口漏单可以提示风格切换。
- 关键结论：盘中看盘的高级目标，是看出这种趋势与投机之间的 PK，而不是事后只知道谁涨谁跌。

## 五、课后应用卡

### 实盘检查清单

- [ ] 9:25 后，一字板分别属于哪些板块，封单比前一日加强还是减弱？
- [ ] 开盘后是否出现与一字方向呼应的快速自然涨停？
- [ ] 板块涨幅榜显示今天主攻方向是什么？
- [ ] 板块涨速榜显示当下资金正在回流哪里？
- [ ] 短线精灵和个股涨速榜是否与板块涨速榜互相印证？
- [ ] 核心票上冲时，后排是否跟随，板块是一致还是分化？
- [ ] 跌速榜里是否出现被吸血的方向？
- [ ] 是否有两个方向多次形成“A 涨 B 跌”的对流关系？
- [ ] 持仓计划是否已经先执行，盘中是否避免只盯持仓？
- [ ] 计划个股是否单独放入观察区，按模式决定做或不做？

### 本课与前后课程的接口

- 前置输入：第 4 课复盘得到的板块核心、计划个股、情绪预期和持仓预案。
- 向后输出：为后续情绪锚定、择时、选股、盘口和具体战法提供盘中观察入口。

### 本课尚未解决的问题

- 情绪锚定票如何在盘中快速确认。
- 弱转强模式在量化套利环境下如何过滤冲高回落。
- 盘口细节如何区分吃货、出货和正常换手。
- 板块吸血关系出现后，如何转化为仓位和买卖动作。
