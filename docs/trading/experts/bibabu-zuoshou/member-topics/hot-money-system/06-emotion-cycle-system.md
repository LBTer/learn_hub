## 一、本课定位与学习目标

- 系统位置：从复盘和看盘工具进入情绪周期判断，把市场环境分成混沌、主升和退潮。
- 前置知识：需要理解连板、涨停跌停、炸板率、断板反馈、龙头、板块联动和主线题材。
- 本课解决：怎样识别不同周期状态，以及不同周期下仓位和操作方向应如何变化。
- 学完能够：用数据和龙头反馈判断市场处于混沌、主升还是退潮，并知道主线题材左侧发现和右侧确认的基本条件。
- 本课不解决：各周期节点的具体买点、首板 / 一进二 / 首分核心 / 反包转强等细化手法，这些作者放到下节课继续讲。

作者反复提醒，情绪周期理论很老，不是学会名词就能直接交易。新手最容易把分歧看成退潮，或把没到高潮的数据误判成高潮，所以应先结合每日计划和盘中交流慢慢校准。

## 二、核心规则与知识结构

### 核心规则

1. 周期要用数据量化：连板数量、涨停家数、跌停家数、炸板率、断板反馈和大幅回撤数量要一起看。
2. 混沌期的关键不是“没机会”，而是没有明确主线、题材轮动快、空间高度打不开。
3. 主升期必须有明确主线和龙头，板块个股集体联动，赚钱效应明显大于亏钱效应。
4. 退潮不是普通分歧，只有主升大分歧后不能修复、周期龙跌停或高位持续补跌，才更接近退潮确认。
5. 不同周期对应不同仓位：混沌小仓试错，主升重仓主线，退潮空仓或极小仓博弈补涨 / 穿越。
6. 主线题材至少要具备龙头、容量中军、20 厘米弹性和龙头属性发散；满足四要素不一定是主线，但主线必须满足四要素。

### 知识结构

![情绪周期总览](/images/trading/bibabu/member-hot-money-system/06/0060.jpg)

**视频时间点：01:00**

```text
情绪数据
→ 连板、涨停、跌停、炸板、断板反馈、回撤
→ 判断周期状态

混沌期：轮动快 + 空间打不开 + 小仓试错
主升期：主线明确 + 龙头带队 + 赚钱效应强 + 重仓主线
退潮期：分歧后不能修复 + 周期龙负反馈 + 空仓优先

主线识别
→ 左侧：板块最强、涨停和连板多、新题材、强逻辑、持续消息
→ 右侧：分歧扛跌、修复带头
```

## 三、决策流程与使用边界

### 判断周期：先找主升和退潮，剩下归入混沌

- 观察信号：主线是否明确、龙头是否有号召力、连板和空间是否打开、跌停和炸板是否失控、断板反馈是否恶劣。
- 判断条件：能确认主线和龙头，通常就是主升；周期龙跌停、高位补跌、分歧后无法修复，更接近退潮；两者都不明显且情绪不算崩，就按混沌处理。
- 对应动作：混沌期用低位小仓试错；主升期围绕主线和龙头；退潮期把防守放在第一位。
- 常见误用：看到一次分歧就喊退潮，或看到板块强一天就判断主升。
- 失效边界：周期在事后才能完全定义，事中只能做概率判断和预案修正。

### 混沌期：空间打不开时只做低位试错

![混沌期特征与高度压制](/images/trading/bibabu/member-hot-money-system/06/0330.jpg)

**视频时间点：05:30**

- 观察信号：板块无序轮动，最高空间长期压制在四五板附近，连板晋级率低，高位股容易跌停。
- 判断条件：没有连续强势板块，也没有真正能带队的核心票。
- 对应动作：博弈低吸、低位首板和一进二，仓位最多四成，做错认错，冲高就卖。
- 常见误用：在混沌期做高位，或者把轮动中的单日强板块当成主线。
- 失效边界：现在很多票走断板反包和二波，不能只看连板高度，要看总体空间高度。

### 主升期：确认主线后只围绕最强方向

- 观察信号：主线明确、板块共涨共跌、龙头能带队上攻、板块成交量放大、赚钱效应明显大于亏钱效应。
- 判断条件：龙头打开空间并成为风向标，做错也有容错，炸板股次日不给严重负反馈。
- 对应动作：确认主升和龙头后提高仓位，龙头买不到就找中军和弹性，原则上只看主线不看支线。
- 常见误用：支线也有强度时分散注意力，导致主线行情里没有拿到最强收益。
- 失效边界：主升中也会有跌停和亏钱点，不能要求市场百分百没有亏钱，只看赚钱效应是否明显大于亏钱效应。

### 退潮期：分歧修复失败后防守优先

![退潮期与分歧修复关系](/images/trading/bibabu/member-hot-money-system/06/0900.jpg)

**视频时间点：15:00**

- 观察信号：主线大分歧后接不住，领涨题材资金流出，周期龙跌停，高位股补跌，市场高度连续下降。
- 判断条件：不是第一次分歧就退潮，而是分歧后不能修复，周期龙和高位反馈持续恶化。
- 对应动作：空仓优先，最多低仓小仓博弈补涨或穿越。
- 常见误用：退潮期还想沿用主升期仓位，或者把补涨和穿越当成高胜率机会。
- 失效边界：补涨和穿越本质都是博弈，作者明确说胜率很低。

### 主线识别：左侧找强，右侧看扛跌和带头

![主线四要素与商业航天案例](/images/trading/bibabu/member-hot-money-system/06/1320.jpg)

**视频时间点：22:00**

- 观察信号：龙头、容量中军、20 厘米弹性、龙头属性发散，板块涨停数量和连板家数。
- 判断条件：左侧看板块是否最强、新题材是否好于旧题材、逻辑和消息是否持续；右侧看分歧时是否扛跌、修复时是否主动领涨。
- 对应动作：满足四要素的板块加大关注，但仍要继续验证逻辑、消息和后续反馈。
- 常见误用：满足四要素就直接认定主线。作者强调“满足四要素不一定是主线，但主线一定满足四要素”。
- 失效边界：旧的“共振大盘”说法在当前强指数环境下不能机械套用。

## 四、典型案例与对照

| 案例 | 教学作用 | 类型 | 关键结论 |
| --- | --- | --- | --- |
| 合富中国 | 展示周期龙、空间压制、退潮确认和修复带动 | 主案例 | 周期龙跌停会拖累短线情绪，反包无负反馈后跌停减少 |
| 航天发展 | 展示穿越和商业航天主线结构 | 主案例 | 在合富中国杀跌时抗跌上行，作者视作穿越案例 |
| 商业航天 | 展示主线四要素 | 主线案例 | 有龙头、容量、弹性和属性发散，满足主线候选基础条件 |
| 海峡方向 | 展示右侧确认主线 | 对照案例 | 分歧时扛跌，修复时由合富中国带头上板 |
| 中水渔业 | 展示补涨与退潮高度下降 | 补涨案例 | 补涨核心断板后，高度连续下降，退潮特征更明显 |

### 主案例：合富中国作为周期龙

- 案例背景：作者把合富中国定义为当期周期龙，因为它打开连板和空间高度，周期内没有主板票空间超过它。
- 作者拆解：合富中国跌停后，市场短线情绪跟着恶化；合富中国再次跌停时，全市场也会陪葬；等它真正没有负反馈并反包，跌停数量就明显减少。
- 对应规则：周期龙不只是一个强势股，而是周期情绪的风向标。
- 关键结论：周期龙跌停常是退潮确认信号，周期龙修复则能显著改善短线情绪。

### 主案例：航天发展作为穿越与商业航天龙头

- 案例背景：航天发展启动于合富中国主升期，但合富中国杀跌时，航天发展没有跟随杀跌，而是相对抗跌并向上。
- 作者拆解：正常同周期票应跟随周期龙退潮，少数票不跟随杀跌并向上，就是穿越。
- 对应规则：穿越需要观察它是否脱离原周期龙负反馈。
- 关键结论：作者认为航天发展是穿越，不是凭感觉，而有周期理论支撑。

### 主线案例：商业航天的四要素

- 案例背景：作者用商业航天解释主线四要素。
- 作者拆解：航天发展是龙头；成交额前排有百亿级和几十亿级成交，说明容量有；涨幅榜有 20 厘米和高弹性票；后排大量涨停，说明龙头属性发散。
- 对应规则：主线必须有龙头、容量、弹性和属性发散。
- 关键结论：满足四要素的板块值得重点关注，但还要继续看逻辑和消息刺激。

### 对照案例：海峡方向右侧确认

- 对照关系：商业航天更多用于四要素说明，海峡方向用于右侧确认说明。
- 容易误判之处：只看某天强弱，不看分歧和修复过程。
- 作者结论：周五早盘全市场受摩尔线程影响分歧时，合富中国和海峡方向扛跌；修复时合富中国主动领涨、快速上板，这就是主线右侧确认。

## 五、课后应用卡

### 实盘检查清单

- [ ] 今天连板数量、涨停数量、跌停数量和炸板率分别如何？
- [ ] 昨日断板票今天是反包、横住，还是 A 杀、跌停？
- [ ] 最高空间是否连续压制在四五板附近？
- [ ] 当前是否有全市场公认的主线和周期龙？
- [ ] 龙头是否打开空间，并且能带动板块和短线情绪？
- [ ] 主线分歧后是否能修复，还是接不住直接走弱？
- [ ] 高位股是否开始补跌，市场高度是否连续下降？
- [ ] 当前操作应归为混沌试错、主升进攻，还是退潮防守？
- [ ] 候选主线是否具备龙头、容量、弹性和属性发散？
- [ ] 分歧时谁扛跌，修复时谁主动带头？

### 本课与前后课程的接口

- 前置输入：第 4 课复盘中的情绪、高标、断板反馈、板块和个股记录；第 5 课盘中看盘中的板块和核心票反馈。
- 向后输出：为后续“情绪锚定”和不同周期节点下的手法选择提供市场环境判断。

### 本课尚未解决的问题

- 如何精确锚定主升初期、首次分歧、加速、二次强分歧、退潮和反弹节点。
- 首板、一进二、前排卡位、首分核心、弱转强、反包转强分别如何筛选。
- 妖股和核心龙头的判断标准。
- 补涨和穿越的胜率、仓位和退出条件如何量化。
