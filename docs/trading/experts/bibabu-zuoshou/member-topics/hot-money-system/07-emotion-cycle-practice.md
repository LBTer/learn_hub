## 一、本课定位与学习目标

- 系统位置：承接第 6 课情绪周期理论，把混沌、主升、退潮落到题材节奏表和具体操作节点。
- 前置知识：需要理解题材复盘表、情绪周期、主线四要素、首板、一进二、分歧、弱转强、龙头和核心票。
- 本课解决：怎样用题材作为情绪周期的载体，识别主线诞生、首次大分歧、主升二、龙头切换和退潮空仓。
- 学完能够：用题材节奏表追踪“题材启动 → 低位试错 → 首次大分歧 → 核心转强 → 龙头切换 → 退潮”的过程，并知道不同节点仓位和动作不同。
- 本课不解决：弱转强的细节买点、择股模型的完整规则、周期延续的量化确认，这些作者放到后续课程或实战中继续讲。

作者这一课的核心意思很直接：情绪周期不能只停在“混沌、主升、退潮”的名词上，必须落到题材和题材复盘表上。情绪是外在，题材是载体；不把涨停原因、个股定位和题材节奏整理清楚，就很难知道自己是在试错、进攻还是防守。

## 二、核心规则与知识结构

### 核心规则

1. 情绪周期要落到题材周期上看，题材复盘表是把全市场拆成可观察结构的工具。
2. 混沌期的一进二、二进三本质是小仓试错，目的不是每次都对，而是抓到未来主升龙。
3. 首次大分歧不等于题材结束，要看后排是否仍有强度，以及核心能否弱转强或反包。
4. 核心票至少具备龙头的一两个特性：领涨、主动回流、空间高、扛跌；三性都具备才更接近龙头。
5. 大题材可能发生龙头切换，原龙头负反馈后，新核心若主动回流、扛跌且空间更强，就可能接力成为新龙头。
6. 退潮期要第一时间定义并空仓，周期理论最大的价值之一是避免在退潮期回撤。

```text
→ 主线题材提高仓位

→ 普通题材多在这里结束
→ 大题材可能龙头切换或进入主升三

退潮
→ 龙头跌停 / 高位负反馈
→ 空仓优先
```

## 三、决策流程与使用边界

### 用题材复盘表把盘面结构化

- 观察信号：全天涨停原因、题材归类、每个题材内部的身位、龙头、中军、容量、弹性、后排、核心。
- 判断条件：题材是否从弱势轮动中胜出，是否出现启动日表态票、身位票和持续后排助攻。
- 对应动作：每天更新题材节奏，不只看单只票涨停，而是看它在题材里的定位和题材整体节奏。
- 常见误用：只看一个票强，不看题材是否仍有后排和资金承接。
- 失效边界：题材复盘表不是买卖指令，它只能帮助锚定结构；具体买点还要结合竞价、分歧、回流和仓位。

### 混沌期试错：小仓做未来主升龙的可能性

- 观察信号：旧周期退潮后，新题材还没确认，低位首板、一进二、二进三开始出现。
- 判断条件：还看不清谁是主线，但某个票先于板块启动，或启动日表态明显。
- 对应动作：用小仓试错，作者举例 0.5 成、1 成；激进选手可以更高，但本质仍是试错。
- 常见误用：试错失败后不认错，或者把试错仓位当主升仓位用。
- 失效边界：如果低位试错的上方空间不够，盈亏比不支持反复失败，就不能硬做。

### 首次大分歧：前排断板时先看后排还有没有强度

![首次大分歧后的核心选择](https://learn.luovista.com/trading/bibabu/member-hot-money-system/07/1460.jpg)

**视频时间点：24:20**

- 观察信号：前排连续一字或晋级后开始放量、断板，板块内部涨停数量明显减少。
- 判断条件：如果后排仍有涨停和强度，说明资金没有彻底结束；如果板块一个涨停都没有，才更接近结束。
- 对应动作：分歧日只看最核心、最扛跌的票。买不到更强核心时，才退而求其次看能交易的核心。
- 常见误用：前排一断板就判断题材结束，或分歧日去吸杂毛后排。
- 失效边界：首次大分歧是头铁买点，不是确定性买点；次日不能弱转强，就要承认判断失败。

### 龙头切换：二次分歧后看谁主动接力

![龙头切换到合富中国](https://learn.luovista.com/trading/bibabu/member-hot-money-system/07/1625.jpg)

**视频时间点：27:05**

- 观察信号：原龙头出现负反馈，板块未彻底退潮，另一只票在分歧中更扛跌、回流更主动、空间更高。
- 判断条件：新核心不仅自己走强，还能把板块带起来，形成二次分歧后的转强。
- 对应动作：把它视作新龙头候选，重新围绕主线核心观察，而不是死盯原龙头。
- 常见误用：把所有卡位都当龙头切换，忽视它是否带板块。
- 失效边界：不能带板块的独立走强更接近妖股，不等于周期龙。

### 退潮期：先定义，再保护利润

- 观察信号：龙头跌停、高位负反馈、主线接不住、题材节奏表显示板块整体走弱。
- 判断条件：周期从主升进入退潮后，继续做原来的进攻动作容易造成回撤。
- 对应动作：空仓优先，不在退潮期硬找机会。
- 常见误用：退潮期仍用主升仓位，或者把反弹当新主升。
- 失效边界：退潮后的反弹可以博弈，但需要有合富中国、平潭发展这种明确照亮的核心，且再次跌停是结束信号。

## 四、典型案例与对照

| 案例 | 教学作用 | 类型 | 关键结论 |
| --- | --- | --- | --- |
| 平潭发展与海峡概念 | 展示低位试错、启动日定位、首次大分歧和主升二 | 主案例 | 一进二不一定带板块，但试错成功后能躺到主升；首次分歧要看后排强度 |
| 合富中国 | 展示龙头切换和大周期成立 | 主案例 | 原龙头负反馈后，合富中国主动回流、扛跌且空间更高，完成新龙头确认 |
| 航天发展 | 展示穿越不用死抠定义 | 方法案例 | 扛跌 + 主动回流就是核心特征，不必先争论是否叫穿越 |
| 海马汽车 | 展示主线和分支同样弱转强但结果不同 | 对照案例 | 有主线做主线，分支弱转强要谨慎；主升环境会提供容错 |
| 摩恩电气 / 中水渔业 | 展示普通小题材主升较短 | 对照案例 | 多数题材主升一、主升二就结束，不要默认都有主升三 |

### 主案例：平潭发展从试错到首次大分歧

- 案例背景：海峡题材启动早期，平潭发展先于后排启动，一板、二板时还不明显带板块。
- 作者拆解：低位一进二本来就是试错，不需要当时就看清主线。10 月 27 日题材爆发时，复盘表把平潭发展标成身位一字，后排有厦门港务、海峡创新、福建水泥、漳州发展等。
- 对应规则：启动日表态和身位票是早期锁定龙头的重要线索。
- 关键结论：试错的意义是用小亏损换一次主升龙机会；成功后，第一次大分歧能否修复要看后排是否仍有强度。

### 主案例：合富中国完成龙头切换

- 案例背景：平潭发展主升二后出现负反馈，作者原本按普通小周期看待，认为周期可能结束。
- 作者拆解：二次分歧后，转强的不是平潭发展，而是合富中国。合富中国下午卡位上板、反包，带动板块回流，成为新的龙头锚点。
- 对应规则：龙头切换要同时看主动性、扛跌性和空间高度。
- 关键结论：海峡概念能走成大周期，不是因为原龙头一直强，而是二次分歧后有新核心接力。

### 方法案例：航天发展不必先定义是否穿越

- 案例背景：作者提到航天发展在合富中国、平潭发展下杀时扛跌，之后又主动回流。
- 作者拆解：与其纠结它是不是穿越，不如直接观察它是否具备扛跌和主动回流。
- 对应规则：核心票至少具备龙头的一两个特性。
- 关键结论：短线择股要抓可观察特征，而不是先陷入概念命名。

### 对照案例：海马汽车弱转强为什么不如平潭发展

- 对照关系：海马汽车和平潭发展都出现弱转强手法，但平潭发展属于海峡主线，海马汽车更像分支。
- 容易误判之处：只看形态都是弱转强，却忽略主线和分支的周期地位。
- 作者结论：有主线做主线，分支题材的弱转强要小心。主升期赚钱效应强，分支失败也可能有容错，但这不代表分支优先级高。

## 五、课后应用卡

### 实盘检查清单

- [ ] 今天的涨停是否已经按题材原因归类？
- [ ] 每个题材里是否标清身位、板块龙、中军、容量、弹性、后排和核心？
- [ ] 当前市场是混沌、主升、退潮，还是退潮后的反弹？
- [ ] 新题材是否处在首板、一进二、二进三的小仓试错阶段？
- [ ] 启动日表态票是谁，是最先上板、封单最大，还是先于板块启动的身位票？
- [ ] 首次大分歧时，前排断板后后排还有没有涨停和强度？
- [ ] 分歧日最扛跌、最主动回流的核心是谁？
- [ ] 弱转强后，是主线题材还是分支题材？
- [ ] 二次分歧后是否出现龙头切换，新核心是否带板块？
- [ ] 龙头跌停或主线接不住时，是否已经按退潮处理并降低仓位？

### 本课与前后课程的接口

- 前置输入：第 4 课的复盘工具和题材分类，第 5 课的竞价与盘中回流观察，第 6 课的混沌、主升、退潮理论。
- 向后输出：为后续择股课提供“领涨、主动、扛跌、空间高”的核心票特征；为弱转强课提供首次分歧、二次分歧和反包节点。

### 本课尚未解决的问题

- 弱转强的具体盘口、竞价和上板确认条件。
- 龙头切换当天如何区分真切换和短线卡位套利。
- 分支题材弱转强在什么环境下可以做，应该给多少仓位。
- 周期延续和复杂顶如何量化识别。
