## 一、本课定位与学习目标

- 系统位置：承接第 6、7 课的情绪周期和题材节奏，把“主流题材里做核心票”进一步落实为可观察的选股信号。
- 前置知识：需要理解主流题材、题材节奏表、核心票、龙头、分歧、修复、竞价、涨速、区间涨幅和常见交易模式。
- 本课解决：如何从强势榜、竞价、盘中涨速、赚钱效应、独立逻辑和个人模式里筛出更接近市场核心审美的票。
- 学完能够：先判断是否适合做游资强势股，再用“题材优先 → 龙头三性 → 五类选股工具 → 模式过滤”的顺序建立核心票池。
- 本课不解决：低吸埋伏的完整规则、竞价专题细节、弱转强买点、具体买卖和仓位执行。

作者这一课的核心提醒是：择股不是孤立看“这个股好不好”，而是先看择时、题材和赚钱效应，再从板块里找最有领涨、带动、扛跌特征的核心票。选股工具本身只是“术”，真正决定方向的是主流题材和当前赚钱效应。

## 二、核心规则与知识结构

### 核心规则

1. 游资强势股择股适合超短线，不适合机械套到低吸、埋伏或中军模式上。
2. 个股选择前先选板块，优先主流题材，因为主流题材有更高溢价和容错率。
3. 核心票至少要具备龙头三性中的一两个：领涨、带动、扛跌；三性都具备才更接近龙头。
4. 区间涨幅、竞价金额、竞价换手、涨速榜都是工具，必须结合题材状态和分歧 / 修复环境使用。
5. 赚钱效应择股是最核心的方法：每个周期赚钱的图形和属性会变，但围绕赚钱效应模仿是不变的。
6. 无模式不买股。中军、连板、低吸、埋伏、一字定方向各有选股逻辑，不能用一个指标硬套所有模式。

### 知识结构

![龙头三性与择股方法导图](https://learn.luovista.com/trading/bibabu/member-hot-money-system/08/0260.jpg)

**视频时间点：04:20**

```text

先择时、看大局
→ 选主流题材
→ 在题材内找核心票

核心票判断
→ 领涨：空间高、涨得早、分歧日回流早
→ 带动：上攻 / 上板时带动后排
→ 扛跌：分歧时红盘、横盘或主动上攻

五类择股工具
→ 区间涨幅：找领涨与独立强势
→ 竞价：看竞价涨幅、金额、换手和板块状态
→ 涨速：消息或回流时找最快响应
→ 赚钱效应：找当前周期最赚钱图形或属性
→ 逻辑 / 强势股：研究题材核心或独立逻辑

最后过滤
→ 是否符合自己的交易模式
→ 是否有市场后续接力的可能
```

## 三、决策流程与使用边界

### 龙头三性：先确认是不是核心票

![西部材料与后排分时叠加](https://learn.luovista.com/trading/bibabu/member-hot-money-system/08/0410.jpg)

**视频时间点：06:50**

- 观察信号：板块内空间高度、起涨时间、上板时后排是否跟随、分歧时是否横盘或红盘。
- 判断条件：一个票具备一两个特性，可以先视为核心候选；三性都具备，才更接近龙头。
- 对应动作：把这类票加入自选，继续观察它是否影响次日板块走势。
- 常见误用：只看涨幅高，不看带动和扛跌；或只看单票强，不看板块是否认可。
- 失效边界：龙头是市场博弈出来的，选到核心不等于必然选到龙头；核心候选可能暂时不被市场选择。

### 区间涨幅择股：复盘必须知道强势榜前二十做什么

![七日涨幅榜选领涨个股](https://learn.luovista.com/trading/bibabu/member-hot-money-system/08/1075.jpg)

**视频时间点：17:55**

- 观察信号：五日、七日、十日、二十日涨幅榜，尤其是同一票在多个周期都靠前。
- 判断条件：多周期共振靠前，说明空间和强势得到确认；独立票则可能通过十日榜或涨停持续性暴露出来。
- 对应动作：每天给强势榜前二十做概念标注，至少知道它炒什么、是否属于主流题材、是否有后续预期。
- 常见误用：只把涨幅榜当排名看，不研究票的题材、逻辑和赚钱效应属性。
- 失效边界：低位一进二、二进三不适合单纯按空间排序找票；这种阶段还需要其他试错逻辑。

### 竞价择股：竞价强要放进分歧 / 修复语境

![竞价金额、竞价换手和竞价涨幅](https://learn.luovista.com/trading/bibabu/member-hot-money-system/08/1265.jpg)

**视频时间点：21:05**

- 观察信号：竞价涨幅、早盘竞价金额、竞价换手、早盘未匹配金额、前一天板块状态。
- 判断条件：竞价金额是绝对比较，竞价换手是相对比较；分歧日与修复日的解释不同。
- 对应动作：分歧转一致时，优先找竞价涨幅高、竞价金额大、与板块状态匹配的核心票。
- 常见误用：把竞价强直接等同于开盘强，忽略开盘兑现和板块不确认。
- 失效边界：分歧日如果竞价量能过大，可能是里面资金兑现，而不是新增资金看好。

![雪人集团竞价择股案例](https://learn.luovista.com/trading/bibabu/member-hot-money-system/08/1455.jpg)

**视频时间点：24:15**

作者用雪人集团说明：前一天核电板块分歧，第二天竞价涨幅和竞价金额都排在前面，就可以理解为分歧转一致的候选。这个例子不是单看竞价高，而是竞价信号和板块节奏合在一起。

### 涨速择股：盘中消息后找最先响应者

- 观察信号：同花顺弹窗、盘中利好、板块快速回流、涨速榜排名。
- 判断条件：涨得最快的票是否与消息或板块利好一致，是否最先上板并封住。
- 对应动作：盘中用全部 A 股涨速排序快速定位最先响应的票，再回看所属板块和题材逻辑。
- 常见误用：看到涨速快就追，不确认消息是否对应、是否只是冲高回落。
- 失效边界：涨速择股更适合盘中动态观察，静态复盘只能保留方法，无法替代当时盘口。

### 赚钱效应择股：每个周期围绕最赚钱图形模仿

![平台二波赚钱效应](https://learn.luovista.com/trading/bibabu/member-hot-money-system/08/1835.jpg)

**视频时间点：30:35**

- 观察信号：最近涨停票、强势票的共同图形和共同属性，例如平台二波、反包、地天、连板、N 字板、次新、小市值、低价或玄学。
- 判断条件：当前最赚钱的图形和属性是什么，市场是否反复模仿这种图形。
- 对应动作：选股时往当前赚钱效应靠。例如本课中，作者认为当时最强的是平台二波，就把西部材料、航天机电等放进这个框架观察。
- 常见误用：只学固定图形，不每天更新当前周期真正赚钱的东西。
- 失效边界：赚钱效应会变，上一周期有效的图形，换到下一周期可能失效。

作者认为这是本课最核心的地方。很多票选不出来，是因为没有抓住最近赚钱效应；一旦抓住，就知道自己想要什么样的票。

### 无模式不买股：工具必须服从交易模式

![择股方法与模式适配](https://learn.luovista.com/trading/bibabu/member-hot-money-system/08/2115.jpg)

**视频时间点：35:15**

- 观察信号：自己的模式是中军、连板、低吸、埋伏、一字定方向，还是游资强势股。
- 判断条件：指标是否真的服务于该模式。中军看成交量和图形流畅度，连板看题材、卡位、图形和筹码，埋伏看吸筹、筹码沉淀和催化预期。
- 对应动作：只买符合自己模式的票；竞价、区间涨幅、涨速、人气榜都作为辅助，不替代模式本身。
- 常见误用：用竞价换手筛中军，用区间涨幅筛低位埋伏，或者没有竞争关系的两只票硬比竞价金额。
- 失效边界：工具不匹配模式时，指标再漂亮也可能没有交易意义。

## 四、典型案例与对照

| 案例 | 教学作用 | 类型 | 关键结论 |
| --- | --- | --- | --- |
| 西部材料 | 展示领涨、带动、扛跌和平台二波如何叠加 | 主案例 | 扛跌只是术，真正支撑选择的是商业航天核心地位和平台二波赚钱效应 |
| 航天机电 | 展示核心候选不一定立刻被市场选择 | 对照案例 | 选到核心候选不等于选错，只是市场暂时可能选择另一个更强核心 |
| 雪人集团 | 展示竞价涨幅、金额和板块分歧转一致的结合 | 主案例 | 竞价选股不能脱离前一天板块状态和分歧 / 修复语境 |
| 浙江世宝 | 展示带动性和热度如何让一只票成为板块核心 | 方法案例 | 连续大长腿带动自驾板块超预期，后续复盘必须纳入核心观察 |
| 航天动力 / 西部材料 | 展示逻辑研究找题材链条核心 | 方法案例 | 题材最核心票可以来自供应关系、业务逻辑和市场最认可的炒作线索 |
| 杂毛股与核心股 | 展示为何不做弱跟随 | 反例 | 杂毛分歧时跌、修复时不修复，核心票分歧抗跌、修复先回流 |

### 主案例：西部材料从强势榜到赚钱效应

- 案例背景：商业航天方向里，西部材料在五日、十日、二十日涨幅排序中多次靠前，周四、周五表现亮眼。
- 作者拆解：它既能通过区间涨幅筛出来，也能在分歧日表现出扛跌；某些日子上板时还能带动后排，具备核心票特征。
- 对应规则：领涨、带动、扛跌可以让它进入核心候选，但作者认为更深层的原因是平台二波赚钱效应。
- 关键结论：选股不能停在“这天扛跌”上，要继续问它是否符合当前市场最赚钱的图形和题材审美。

### 主案例：雪人集团的分歧转一致竞价

- 案例背景：核电板块前一天分歧，第二天竞价阶段需要找是否有分歧转一致的票。
- 作者拆解：雪人集团在主板竞价涨幅排序靠前，早盘竞价金额也靠前，结合板块前一日分歧，可以视为分歧转一致候选。
- 对应规则：竞价涨幅、竞价金额、竞价换手要结合板块状态，而不是孤立使用。
- 关键结论：竞价强只是候选信号，开盘后还要看板块和个股能否确认。

### 对照案例：航天机电为什么不是“选错”

- 对照关系：同一阶段，西部材料被市场选择得更强，航天机电也有扛跌和平台二波特征，但没有立刻走出来。
- 容易误判之处：只用结果倒推，认为没涨最多的核心候选就是选错。
- 作者结论：龙头是博弈出来的，做核心票不一定每次买到龙头；市场暂时没有选择，不代表选股逻辑完全错误。

### 方法案例：赚钱效应属性二选一

- 案例背景：不同阶段会出现低价、小市值、次新、名字玄学等属性赚钱效应。
- 作者拆解：当 10 元以下低价股占涨停大多数，同样两只票二选一时，低价优先；当小市值赚钱效应更好，50 亿和 500 亿二选一时，小市值优先。
- 对应规则：除了图形，属性本身也可能成为阶段性赚钱效应。
- 关键结论：选股要先知道当前市场想买什么，再在同类票里做取舍。

## 五、课后应用卡

### 实盘检查清单

- [ ] 今天是否已经先判断市场和题材，而不是直接看个股？
- [ ] 当前主流题材是谁，是否有持续赚钱效应？
- [ ] 题材内哪些票具备领涨、带动、扛跌？
- [ ] 五日、七日、十日、二十日涨幅榜前列分别是什么票？
- [ ] 强势榜前二十是否都标注了题材、逻辑和赚钱效应属性？
- [ ] 竞价强的票，是分歧转一致，还是一致后兑现？
- [ ] 竞价金额是绝对强，还是相对流通市值其实一般？
- [ ] 盘中消息出现后，涨速最快的票是否与消息板块一致？
- [ ] 最近三天涨停票的共同图形是什么：平台二波、反包、连板、N 字、低价、小市值还是其他？
- [ ] 当前要买的票是否符合自己的模式，而不是只符合某个单项指标？
- [ ] 这只票是否能先于市场审美进入自选，后续是否可能有人接力？

### 本课与前后课程的接口

- 前置输入：第 4 课的复盘流程，第 5 课的竞价、涨速和界面设置，第 6、7 课的情绪周期、主流题材和题材节奏表。
- 向后输出：为后续竞价专题、弱转强、买点卖点和具体战法提供核心票池筛选方法。

### 本课尚未解决的问题

- 分歧日竞价换手和修复日竞价金额的具体量化阈值。
- 如何在盘中确认涨速票不是冲高回落。
- 平台二波、反包、N 字、连板之间的优先级如何随周期切换。
- 逻辑研究与盘面确认发生冲突时如何取舍。
- 同一题材多个核心候选之间，如何进一步确定仓位和买点。
