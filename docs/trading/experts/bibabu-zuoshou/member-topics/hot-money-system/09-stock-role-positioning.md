## 一、本课定位与学习目标

- 系统位置：承接第 7 课题材节奏表和第 8 课强势股择股，把“核心票”进一步拆成龙头、情绪、身位、中军、弹性、容量、趋势、跟风和后排杂毛等可标注角色。
- 前置知识：需要理解龙头三性、主流题材、题材节奏表、涨停复盘表、竞价封单、连板、20 厘米 / 30 厘米和一字定方向。
- 本课解决：板块启动后，如何在复盘表里把每只前排票放到正确位置，避免只说“这个票好”。
- 学完能够：先定位情绪票和身位票，再用成交额、涨幅、代码、图形和主动性筛出中军、容量、弹性、趋势和最强跟风，最后用排除法剔除后排杂毛。
- 本课不解决：龙头专题的完整量化规则、补涨挖掘专题、具体买点和仓位分配。

作者这一课的核心提醒是：题材初期不要幻想一二板就定龙头，而要把核心角色都选出来观察。龙头是从核心票里走出来的，交易上应先有“核心思维”，再等待市场选择。

## 二、核心规则与知识结构

### 核心规则

1. 龙头三性仍是底层判断：领涨、抗跌、主动，但题材初期更重要的是把核心候选都找出来。
2. 情绪龙看逻辑和封单，身位龙看是否先于节点启动、是否高一个身位。
3. 中军和容量主要看市值、逻辑和成交额；当前大成交环境下，中军套利效应减弱，但容量仍能说明资金承载能力。
4. 弹性是 20 厘米、30 厘米等大波动品种，适合在前排情绪和后排强度都高时做套利观察。
5. 趋势票沿五日线稳步攀爬，但一旦量能缩小、斜率陡升进入加速，往往接近风险段。
6. 后排杂毛用排除法定义：不是龙头、中军、弹性、容量、趋势、最强跟风等核心角色，就不应作为重点。

```text
板块启动 / 题材发酵
→ 先找情绪票：逻辑最硬、封单最大、能带动情绪
→ 再找身位票：先于节点启动，高一个身位
→ 看成交额：中军 / 容量是否绕不开

交易思维
→ 不在一二板定龙
→ 先建立核心票池
→ 等市场通过领涨、抗跌、主动性选择真正龙头
```

## 三、决策流程与使用边界

### 情绪龙与身位龙：先确定板块前排的两类辨识度

![情绪龙与身位龙示例](https://learn.luovista.com/trading/bibabu/member-hot-money-system/09/0130.jpg)

**视频时间点：01:30**

- 观察信号：封单、逻辑硬度、是否带动情绪、是否先于板块节点启动、连板身位是否更高。
- 判断条件：逻辑最硬且买不到的前排，偏情绪龙；早于消息或板块启动、高一个身位的票，偏身位龙。
- 对应动作：复盘时把情绪和身位单独标出，不要和普通后排混在一起。
- 常见误用：把所有连板都叫龙头，或者只看身位高，不看它是否真的能带动板块。
- 失效边界：情绪票一旦开板，可能已经是板块大分歧；身位票如果逻辑太弱，也可能只被情绪短暂反推。

作者用机器人方向说明：锋龙股份和天奇股份是优必选发酵里的逻辑硬票，偏情绪；大业股份先于节点、有市场化身位，偏身位。智能驾驶里的三羊马，也是复盘表里一眼能看到的身位辨识度。

### 核心思维：龙头从核心票里走出来

![核心思维与龙头思维](https://learn.luovista.com/trading/bibabu/member-hot-money-system/09/0755.jpg)

**视频时间点：07:55**

- 观察信号：个股是否至少具备龙头三性中的两项，是否在板块里绕不开。
- 判断条件：具备两个特性可以先视为核心，三性极致后才更接近龙头。
- 对应动作：持续做和跟踪板块核心票，而不是等待唯一龙头完全确认后才反应。
- 常见误用：极端“不是龙头不买”，导致混沌期完全失去盘感；或把普通强势票过早吹成龙头。
- 失效边界：核心候选不等于最终龙头，市场可能暂时或最终选择另一只票。

作者不赞成普通交易者只做“龙空龙”。真正的龙空龙可能三个月才出手一次，大多数人很难保持盘感。更现实的做法，是每个板块都找核心票，等市场把其中一个推成龙头。

### 中军、容量与弹性：按成交额和涨跌幅定位套利角色

![弹性定义与机器人案例](https://learn.luovista.com/trading/bibabu/member-hot-money-system/09/1235.jpg)

**视频时间点：12:35**

- 观察信号：市值、成交额、板块成交排名、20 厘米 / 30 厘米涨停、是否唯一弹性涨停。
- 判断条件：成交额显著大、能容纳资金的是中军 / 容量；涨幅制度更大、波动更大的，是弹性。
- 对应动作：情绪票买不到时，观察最快换手板、容量 / 中军或弹性票的套利机会。
- 常见误用：沿用旧市场“中军节奏等于板块节奏”的说法，不看当前成交和量化环境变化。
- 失效边界：当前大成交环境下，很多票都能容纳资金，中军影响力可能被稀释；弹性套利也必须依赖前排情绪继续加强。

作者认为中军套利目前有点过时，但概念仍有参考。机器人里的三花智控既是中军也是容量；昊志机电作为 20 厘米涨停，可以放入弹性观察。

### 趋势与加速：趋势票的终点常常是加速风险

![趋势和加速风险](https://learn.luovista.com/trading/bibabu/member-hot-money-system/09/1620.jpg)

**视频时间点：16:20**

- 观察信号：五日线稳步攀爬、均线多头、换手温和、机构或强逻辑主导。
- 判断条件：趋势核心如果开始量能缩小、斜率明显变陡，就进入加速。
- 对应动作：趋势阶段可以作为核心观察；加速阶段尤其后半段，不宜轻易接。
- 常见误用：把加速当作趋势更强，忽略“龙头只会死于加速”的风险。
- 失效边界：不是所有加速都立刻见顶，但后半段连续加速的风险收益比会快速变差。

作者用中国卫星、航天发展、圣龙股份、大众交通等案例说明：分歧不一定杀死龙头，连续加速更容易让龙头见顶。加速不只看涨得快，也看量能缩小和上升斜率陡增。

### 跟风、选美理论与市场审美

![跟风与选美理论](https://learn.luovista.com/trading/bibabu/member-hot-money-system/09/1830.jpg)

**视频时间点：18:30**

- 观察信号：题材相同、代码相近、图形相似、名称相似，或市场主动选择某个非主观最优票。
- 判断条件：买不到最强情绪票时，跟风 / 补涨会沿着“相似性”模仿；但最终以市场选择为准。
- 对应动作：尊重盘面，不因主观不喜欢题材、图形或逻辑而否认市场核心。
- 常见误用：只按自己审美选票，市场已经选出核心仍不跟随。
- 失效边界：模仿票只是跟风，空间和持续性一般弱于龙头，不能把跟风当成真正龙头。

作者借“选美理论”说明：市场选出来的，哪怕不符合自己的主观看法，也要尊重。股票里也是一样，龙头和核心不是由个人喜好决定，而是由市场资金选择。

### 排除后排杂毛：定位完核心角色后，剩余票降级处理

- 观察信号：是否属于龙头、中军、弹性、容量、趋势、最强跟风等角色。
- 判断条件：如果都不是，就统一视为后排杂毛。
- 对应动作：复盘时用排除法减少关注列表，不把每只涨停票都当作可交易对象。
- 常见误用：板块一发酵，就认为所有后排都有机会。
- 失效边界：后排如果隔夜被挖出更硬逻辑，角色可能重估；否则第二天连板概率通常较小。

作者强调，后排套利尽量只在第一天做。第二天再去追换手板，容易变成偏炮套利，当天炸板、次日低开。

## 四、典型案例与对照

| 案例 | 教学作用 | 类型 | 关键结论 |
| --- | --- | --- | --- |
| 机器人：锋龙股份、天奇股份、大业股份 | 区分情绪龙和身位龙 | 主案例 | 锋龙 / 天奇偏逻辑和情绪，大业偏身位 |
| 机器人：三花智控、昊志机电 | 区分中军 / 容量 / 弹性 | 主案例 | 三花是中军和容量，昊志机电是弹性 |
| 智能驾驶：三羊马、北汽蓝谷、万集科技、浙江世宝、索菱股份 | 演示涨停库角色定位 | 主案例 | 身位、中军、弹性、一字前排、最快换手可以逐项标出 |
| 中国卫星、航天发展 | 趋势加速风险 | 方法案例 | 趋势核心加速后，不能只看强，还要看见顶风险 |
| 圣龙股份、大众交通 | 龙头死于加速 | 对照案例 | 连续加速比正常分歧更危险 |
| 福龙马 | 盘中回流主动性 | 方法案例 | 日内最先回流的票可定义为情绪核心 |

### 主案例：机器人板块角色定位

- 案例背景：优必选发酵带动机器人方向，板块内同时出现一字情绪、身位、老龙反包、中军和弹性。
- 作者拆解：锋龙股份和天奇股份逻辑最硬、封单强，偏情绪；大业股份高一个身位，偏身位；首开股份是老龙反包辨识度；三花智控是中军和容量；昊志机电是弹性。
- 对应规则：先找情绪和身位，再按成交额、涨幅制度和历史辨识度补齐角色。
- 关键结论：复盘表不是简单罗列涨停，而是要把每只前排票放到对应角色里。

### 主案例：智能驾驶涨停库定位

- 案例背景：智能驾驶爆发日，作者用涨停库演示如何从表格中选票。
- 作者拆解：三羊马是身位；北汽蓝谷成交额明显大，是中军 / 容量；万集科技是弹性；浙江世宝是一字方向前排；索菱股份是一字后最快换手板。
- 对应规则：身位看连板高度，容量看成交额，弹性看 20 厘米 / 30 厘米，最快换手看一字后的自然涨停速度。
- 关键结论：后排套利主要在第一天，第二天再去追最快换手，容易只吃短暂溢价甚至炸板亏损。

![智能驾驶案例：身位、中军、弹性、换手](https://learn.luovista.com/trading/bibabu/member-hot-money-system/09/2325.jpg)

### 方法案例：趋势核心不要接后半段加速

- 案例背景：商业航天里，中国卫星、航天发展等票走趋势。
- 作者拆解：趋势的最终目的是加速，但加速也意味着风险临近。量能缩小和斜率陡升，都是加速信号。
- 对应规则：趋势票可以跟踪核心地位，但连续加速后要更谨慎。
- 关键结论：分歧本身不一定可怕，龙头和趋势核心更容易死于后半段加速。

## 五、课后应用卡

### 实盘检查清单

- [ ] 今天的板块是否已经先找出情绪票和身位票？
- [ ] 情绪票是逻辑硬、封单大，还是只是普通前排？
- [ ] 身位票是否真的先于节点启动、高一个身位？
- [ ] 板块里成交额最大的票是谁，是否有中军 / 容量意义？
- [ ] 是否有唯一或最先涨停的 20 厘米 / 30 厘米弹性票？
- [ ] 趋势核心是否沿五日线稳步运行，还是已经量缩、斜率加速？
- [ ] 买不到前排时，套利的是最快换手、容量、中军还是弹性？
- [ ] 后排票是否只是排除核心角色后的杂毛？
- [ ] 题材初期是否避免了一二板定龙，而是先建立核心票池？
- [ ] 盘中回流时，谁最主动、最先回流，是否需要提升为情绪核心？

### 本课与前后课程的接口

- 前置输入：第 7 课的题材节奏表、第 8 课的强势股择股方法、第 5 课的盘中监控工具。
- 向后输出：为后续龙头专题、补涨专题、弱转强和具体战法提供板块内角色定位框架。

### 本课尚未解决的问题

- 龙头三性在不同题材阶段如何排序。
- 补涨和跟风如何从题材、代码、图形、名称模仿里进一步筛选。
- 中军套利效应减弱后，哪些市场环境下仍然值得重视中军。
- 多个核心候选并存时，如何进一步转化为买点和仓位。
