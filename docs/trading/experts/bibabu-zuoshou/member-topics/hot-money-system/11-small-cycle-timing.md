## 一、本课定位与学习目标

- 系统位置：承接第 6、7 课情绪周期和题材节奏表，把大周期判断落到具体买点、卖点和不做的节点。
- 前置知识：需要理解混沌、主升、退潮，知道题材节奏表如何记录回流、高潮、分歧和核心票反馈。
- 本课解决：不同大周期下，小周期如何决定买在什么位置、卖在什么位置，以及哪些分歧节点必须放弃。
- 学完能够：先判断大周期，再按小周期识别混沌期买在连续分歧、主升期分歧加仓、退潮期少做或只做反弹。
- 本课不解决：具体选股细节、分歧强弱的量化阈值、主升核心票仓位分配细则。

作者这一课的核心不是“所有分歧都买”，而是先定大周期，再判断分歧的质量。混沌期买在分歧、卖在一致；主升期分歧是加仓机会；退潮期分歧后还可能继续分歧，不能机械低吸。

## 二、核心规则与知识结构

### 核心规则

1. 大周期先行：有周期龙是主升，没有周期龙是混沌，周期龙 A 杀是退潮。
2. 混沌期的小周期是回流、高潮、分歧、分歧延续、再回流；高潮主卖，连续分歧后再博弈回流。
3. 主升期仓位比个股更重要，分歧给买点，但要提高审美，只接核心，不接后排。
4. 退潮期所有买点都可能套人，只有分歧延续后的反弹节点相对可博弈，其他位置多是卖点。
5. 买在分歧是情绪周期的本质，但量化时代小周期加速，必须放弃很多小分歧节点。
6. 四类错误要规避：分歧刚开始接面、主升空仓、冰点割肉、退潮追涨。

### 知识结构

![三种周期的节奏对照](/images/trading/bibabu/member-hot-money-system/11/0450.jpg)

**视频时间点：07:30**

```text
先定大周期
→ 主升：有周期龙，重仓核心，分歧买
→ 混沌：无周期龙，高潮卖，连续分歧后博弈回流
→ 退潮：周期龙 A 杀，少做，最多做超跌反弹

再看小周期
→ 回流：资金回到题材
→ 高潮 / 一致：混沌期偏卖点
→ 分歧：不能立刻机械买
→ 分歧延续：若亏钱效应充分释放，才博弈回流

最终动作
→ 买在分歧，不买在高潮
→ 主升不空仓，退潮不追涨
→ 冰点不割肉，按反弹或做 T 处理
```

## 三、决策流程与使用边界

### 先判断大周期，再决定同一个“分歧”能不能买

![大周期与小周期框架](/images/trading/bibabu/member-hot-money-system/11/0020.jpg)

**视频时间点：00:20**

- 观察信号：是否有市场公认周期龙，周期龙是否 A 杀，题材节奏表顶部的情绪标注。
- 判断条件：有周期龙和共振题材时偏主升；没有龙头、多题材轮动时偏混沌；周期龙跌停或高位亏钱扩散时偏退潮。
- 对应动作：主升期提高仓位，混沌期等连续分歧，退潮期管住手。
- 常见误用：只看到分歧就低吸，不先判断大周期。
- 失效边界：若周期龙、题材核心和市场整体赚钱效应互相冲突，要以盘面反馈继续修正。

### 混沌期：强回流往往就是高潮，高潮主卖

![混沌期回流高潮后接分歧](/images/trading/bibabu/member-hot-money-system/11/0170.jpg)

**视频时间点：02:50**

- 观察信号：某板块强回流、后排批量修复、题材节奏表显示当日明显回流。
- 判断条件：混沌期没有持续主线，强回流后次日大概率分歧。
- 对应动作：高潮日主卖；等分歧和分歧延续后，再找回流预期。
- 常见误用：把混沌期的强回流当成主升，次日继续追高。
- 失效边界：极少数板块可能强回流后延续，但作者认为这是混沌期里的偶然，不应作为默认预期。

### 主升期：分歧是核心票的买点

- 观察信号：周期龙明确，主线题材持续带动，核心票回流后仅小分化。
- 判断条件：小周期不断回流、高潮、分化、再回流，净值曲线整体向上。
- 对应动作：给分歧点就买，核心弱转强、竞价、板上确认、二次分歧都可能是买点。
- 常见误用：仍用混沌期的“高潮次日不接”去处理主升期，导致主升空仓。
- 失效边界：主升期也要提高审美，只接核心，不接后排；周期龙 A 杀后要切换退潮逻辑。

### 退潮期：分歧后还可能继续分歧

- 观察信号：周期龙 A 杀，高位补跌，修复后继续下跌。
- 判断条件：分歧后不是马上回流，而是可能接分歧延续。
- 对应动作：少做、休息；认知强时只做分歧延续后的超跌反弹。
- 常见误用：把退潮第一天的分歧当作低吸点，结果买在分歧截面。
- 失效边界：退潮反弹能否带出来难度很高，不能把反弹当作新主升。

### 量化时代：不是情绪周期失效，而是节奏变快

![量化时代节奏加速](/images/trading/bibabu/member-hot-money-system/11/0960.jpg)

**视频时间点：16:00**

- 观察信号：早盘分歧、日内回流、尾盘反杀在一天内完成。
- 判断条件：小周期循环被压缩，隔日溢价变差。
- 对应动作：放弃小节点，优先选择亏钱效应释放更充分、市场整体性更强的分歧节点。
- 常见误用：认为买点逻辑没错就一定能隔日赚钱。
- 失效边界：若回流只持续半天，次日开盘再分歧时会面临卖不卖的两难。

### 用题材节奏表找“连续分歧后的回流”

![题材节奏表辅助判断](/images/trading/bibabu/member-hot-money-system/11/1220.jpg)

**视频时间点：20:20**

- 观察信号：题材节奏表里某板块昨天分歧、今天继续分歧，核心票仍有可博弈位置。
- 判断条件：若连续分歧释放充分，次日可博弈回流；若只是弱回流后的普通分歧，质量不够。
- 对应动作：在分歧延续低位低吸或做回流预期，失败就止损。
- 常见误用：每天都找分歧节点做，不区分分歧质量。
- 失效边界：选股失败仍会亏钱，本课只解决择时，选股仍要回到第 8 课。

## 四、典型案例与对照

| 案例 | 教学作用 | 类型 | 关键结论 |
| --- | --- | --- | --- |
| 白银有色、湖南白银 | 大周期定义 | 主案例 | 共振上行是金属主升，周期龙 A 杀后进入退潮 |
| AI 玩具、消费 | 混沌期强回流后分歧 | 对照案例 | 强回流次日容易反杀，高潮主卖 |
| 光伏、CPO、电池 | 分歧后回流 | 板块案例 | 已分歧或大分歧后的方向，才更适合博弈回流 |
| 天地在线 | 连续分歧后回流 | 主案例 | 买在分歧延续低位，次日反包 |
| 华光环能、红宝丽、百川股份、双良节能 | 分歧延续低吸 | 方法案例 | 混沌期买在分歧延续，不追高潮 |
| 航天发展 | 选股失败对照 | 反例 | 节点对也可能选错票，失败就处理 |
| 北方铜业、同和药业 | 冰点处理 | 主案例 | 冰点不割肉，回流节点可低吸做 T |

### 主案例：白银有色定义金属周期

- 案例背景：金属板块上行时，白银有色连板向上，湖南白银等共振。
- 作者拆解：有周期龙，就是主升；周期龙跌停，则开始退潮；退潮后没有新龙头，市场回到混沌。
- 对应规则：大周期必须先定，不能只看单日涨跌。
- 关键结论：主升、混沌、退潮对应完全不同的买卖节奏。

### 主案例：天地在线的连续分歧后反包

![连续分歧后的回流博弈](/images/trading/bibabu/member-hot-money-system/11/1380.jpg)

**视频时间点：23:00**

- 案例背景：混沌期里，题材连续分歧后出现低位博弈机会。
- 作者拆解：买在分歧延续的低位，而不是买在回流高潮。
- 对应规则：混沌期核心买点在连续分歧之后。
- 关键结论：低吸或地板附近试错，才更符合混沌期节奏。

### 对照案例：华光成功，航天发展失败

- 对照关系：同样是做分歧回流，选到华光环能、红宝丽、百川股份等方向可能赚钱，选到航天发展则可能亏钱。
- 容易误判之处：把择时成功当成交易成功，忽略择股。
- 作者结论：本课只告诉什么时候该买，具体买谁要回到择股课。

### 主案例：北方铜业冰点低吸做 T

![北方铜业低吸做T示例](/images/trading/bibabu/member-hot-money-system/11/1810.jpg)

**视频时间点：30:10**

- 案例背景：作者在北方铜业、同和药业等操作中有被套经历。
- 作者拆解：周二是明确回流节点，冰点位置不应该割肉，而是低吸做 T 或博弈反弹。
- 对应规则：冰点不是割肉点，退潮拉升才不能追。
- 关键结论：冰点割肉和退潮追涨，是节奏错误里最伤账户的两类。

## 五、课后应用卡

### 实盘检查清单

- [ ] 今天是否有市场公认周期龙？
- [ ] 周期龙是否 A 杀，或高位亏钱效应是否扩散？
- [ ] 当前是主升、混沌还是退潮？
- [ ] 题材节奏表里哪些板块刚回流，哪些已经高潮？
- [ ] 哪些板块连续分歧，是否释放充分？
- [ ] 今天的分歧是主升分歧、混沌分歧，还是退潮分歧？
- [ ] 回流是否是市场整体性回流，还是局部弱回流？
- [ ] 是否只是日内半天回流，尾盘已经反杀？
- [ ] 当前计划是在买分歧延续，还是追高潮接力？
- [ ] 如果回流失败，是否提前设定止损或卖出条件？
- [ ] 是否犯了分歧接面、主升空仓、冰点割肉、退潮追涨四类错误？

### 本课尚未解决的问题

- 分歧延续多少天、什么强度才算释放充分。
- 量化时代日内回流反杀后，隔日能否继续博弈的判断标准。
- 主升期接核心、不接后排的仓位和买点细化。
- 计划成功率如何系统化记录并反推市场环境。
