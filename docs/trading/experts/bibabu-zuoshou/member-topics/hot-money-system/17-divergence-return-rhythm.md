## 一、本课定位与学习目标

- 系统位置：把情绪周期和题材复盘进一步落到“板块当天到底是分歧、回流、高潮还是走弱”的盘中判断。
- 前置知识：需要理解核心票、后排、中军、题材复盘表、混沌期、主升期、退潮期和竞价反馈。
- 本课解决：如何定义强弱分歧、强弱回流，如何根据板块节奏做次日预期，如何处理连续分歧后的尾盘核心套利和尾盘抢筹。
- 学完能够：用核心、后排、中军三类反馈判断板块状态，在分歧和回流初期寻找买点，避开一致高潮后的后排接力。
- 本课不解决：每个题材的产业逻辑真假、个股基本面估值，以及所有案例的事后走势统计。

这节课的重点不是背术语，而是把“分歧”“回流”变成可观察的盘面结构。作者反复强调，买点通常在分歧或回流刚确认时，不在一致高潮之后。

## 二、核心规则与知识结构

### 核心规则

1. 判断分歧强弱，要同时看核心、后排和中军：核心深水或跌停、后排区域跌停、中军负反馈，才是强分歧。
2. 判断回流强弱，也要看合力：核心反包前阴线、后排跟随涨停、中军表态，才是强回流。
3. 分化通常是弱分歧：核心不给机会或红盘震荡，后排给机会；正常预期下，分化后容易接真正分歧。
4. 普通小题材常见“三天节奏”：第一天爆发，第二天分化，第三天大分歧；后排第二天走弱就应先处理。
5. 真正主线分歧后若能强回流，必须重视；主线分歧中不只有前排可能是买点，卡位后排也可能成为新前排。
6. 连续分歧后的尾盘买核心，是博弈次日修复的混沌期手法；但尾盘抢筹会提前消耗次日需求。
7. 买票要买在分歧或回流开始，不能买在一致；强更强再强更强属于一致更加一致，通常不适合接力。

### 知识结构

![分歧回流定义框架](/images/trading/bibabu/member-hot-money-system/17/0005.jpg)

**视频时间点：00:05**

```text
板块状态判断
→ 看核心票：深水 / 跌停 / 红盘震荡 / 反包阴线
→ 看后排：区域跌停 / 分化 / 冲高回落 / 跟风涨停
→ 看中军：负反馈 / 震荡 / 反包 / 表态

强分歧
→ 核心深水或跌停 + 后排区域跌停 + 中军负反馈

弱分歧 / 分化
→ 核心红盘震荡或不给机会 + 后排掉队

强回流
→ 核心反包前阴线 + 后排跟随 + 中军表态

弱回流
→ 核心冲高 + 后排冲高回落

交易节奏
→ 爆发 / 高潮后不追一致
→ 分歧中买核心或等右侧回流
→ 连续分歧尾盘可博弈核心修复
```

## 三、决策流程与使用边界

### 先定义当天板块状态

![机器人强回流案例](/images/trading/bibabu/member-hot-money-system/17/0500.jpg)

**视频时间点：08:20**

- 观察信号：核心票是否深水、跌停、红盘震荡或反包；后排是区域负反馈还是跟随；中军是否表态。
- 判断条件：三类角色共振向下是强分歧，共振向上是强回流；只有局部票冲高，则多半只是弱修复。
- 对应动作：强分歧后看核心能否抗住或次日修复；强回流后把题材列为重点观察；弱修复一般偏卖点。
- 常见误用：只看一只个股涨停，就把整个板块定义为强回流。
- 失效边界：如果核心和中军没有配合，后排大面积冲高回落，不能因为个别票强就提高板块级别。

### 普通小题材：按“三天小题材”处理

- 观察信号：第一天消息爆发，第二天只有绝对核心顶住，后排水下开或冲高回落。
- 判断条件：题材逻辑不硬、后续催化少、后排不跟随，通常难以升级。
- 对应动作：若手上不是绝对核心，第二天走弱先卖；第三天防大分歧。
- 常见误用：看到核心一字，就误以为题材已经成主线。
- 失效边界：若分歧后出现强回流，核心、后排、中军重新共振，则需要重新评估题材级别。

### 主线题材：分歧后强回流必须重视

![强回流与中军表态](/images/trading/bibabu/member-hot-money-system/17/1110.jpg)

**视频时间点：18:30**

- 观察信号：预期分歧后，板块早盘给机会，随后核心反包、中军反包、后排跟随。
- 判断条件：持筹者本来以为要亏或只能平走，结果还能赚钱离场，这就是超预期。
- 对应动作：把题材列为潜在主线，分歧中买核心，或右侧确认回流后跟随。
- 常见误用：主线分歧时过早把所有后排都当卖点，忽略卡位和新前排。
- 失效边界：若只是盘中回流，收盘又转弱，不能按收盘强回流处理。

### 连续分歧尾盘买核心：只做逻辑题材和核心

![连续分歧尾盘买核心](/images/trading/bibabu/member-hot-money-system/17/3030.jpg)

**视频时间点：50:30**

- 观察信号：有逻辑的题材连续分歧两天，板块核心尾盘仍有承接。
- 判断条件：混沌期里，非退潮环境下板块连续三天无修复的概率较低。
- 对应动作：尾盘低吸板块核心，博弈次日弱修复或强修复；冲红即可按套利思路处理。
- 常见误用：买杂毛，或在退潮期还幻想连续分歧必有回流。
- 失效边界：板块进入退潮、核心破位、题材逻辑消失时，不用这个手法。

### 尾盘抢筹和离别钩：先手利好，接力利空

![尾盘抢筹的供需解释](/images/trading/bibabu/member-hot-money-system/17/3300.jpg)

**视频时间点：55:00**

- 观察信号：尾盘大量资金提前买入，股价被抢上去；或连板票无量急拉形成离别钩。
- 判断条件：尾盘抢筹会提前消耗次日修复需求，同时增加次日卖出供给。
- 对应动作：已有先手可把抢筹视为当日浮盈；次日接力资金要降低预期。
- 常见误用：把所有尾盘抢筹都当第二天强更强信号。
- 失效边界：逻辑特别强、板块合力很强时，尾盘抢筹后仍可能强更强；但离别钩更偏风险信号。

### 买在分歧和回流开始，不买一致

![左侧分歧和右侧回流买点](/images/trading/bibabu/member-hot-money-system/17/4520.jpg)

**视频时间点：75:20**

- 观察信号：板块竞价分歧、核心给低位机会，或盘中核心上板、后排同步上冲确认回流。
- 判断条件：左侧买点是分歧中博弈回流，右侧买点是回流开始后确认跟随。
- 对应动作：分歧时买核心，回流初期买辨识度；一致高潮次日以兑现为主。
- 常见误用：看到板块已经高潮、前排一字、后排高溢价后才追。
- 失效边界：强更强后再强更强，属于一致更加一致，风险收益不合适。

## 四、典型案例与对照

| 案例 | 教学作用 | 类型 | 关键结论 |
| --- | --- | --- | --- |
| 算力：华胜天成、鸿博股份、电光科技 | 区分后排掉队和真正强分歧 | 对照案例 | 核心没给极端负反馈时，不轻易定义为强分歧。 |
| 机器人强回流 | 展示分歧后强回流为何要重视 | 主案例 | 首个分歧后接强回流的题材，具备潜在主线意义。 |
| 军工：成飞集成、中航成飞 | 展示超预期回流和中军锚定 | 主案例 | 中军反包前阴线、核心带动回流，是强回流确认。 |
| 磁悬浮、纺织 | 展示三天小题材 | 反例 | 小题材第二天后排走弱，第三天防大分歧。 |
| 华为链 / 青岛金王 | 展示连续分歧尾盘买核心 | 手法案例 | 连续分歧后尾盘拿核心，博弈次日修复冲红。 |
| 尾盘抢筹 / 离别钩 | 展示先手和接力的供需差异 | 风险案例 | 抢筹对先手是利好，对次日接力可能是利空。 |

### 主案例：军工分歧后的强回流

- 案例背景：军工周末预期是大分歧，早盘也确实几乎所有票给了机会。
- 作者拆解：成飞集成主动带动回流，中航成飞作为中军反包前一天阴线，后排也跟随，说明不是普通轮动。
- 对应规则：强回流要看核心、中军、后排共振；中军可以锚定板块节奏。
- 关键结论：预期分歧后还能反包，是超预期；但高潮后次日不适合再追一致，二次介入要等分歧。

### 反例：磁悬浮和纺织的三天小题材

- 对照关系：它们都有消息刺激和首日爆发，但后续逻辑、后排强度和持续催化不足。
- 容易误判之处：第二天绝对核心顶一字，会让人以为板块仍强；但后排水下开或冲高回落，已经暴露弱度。
- 作者结论：小题材手里不是绝对核心，第二天就要先处理；第三天防大分歧和补跌。

### 手法案例：连续分歧尾盘买核心

- 案例背景：混沌期里有逻辑的老题材连续分歧，第三天通常有修复预期。
- 作者拆解：尾盘拿板块核心，次日弱修复冲红即可套利；华为链、青岛金王是示例。
- 对应规则：只买核心，不买杂毛；退潮期不用。
- 关键结论：这不是看多题材长期，而是利用板块节奏做修复预期。

## 五、课后应用卡

### 实盘检查清单

1. 今天板块是爆发、高潮、分化、强分歧、弱回流还是强回流？
2. 核心票是深水、跌停、红盘震荡，还是反包前阴线？
3. 后排是区域跌停、冲高回落，还是有一两个跟风涨停？
4. 中军是负反馈，还是反包、放量表态？
5. 这是普通小题材，还是分歧后能强回流的潜在主线？
6. 当前买点是在分歧、回流开始，还是已经一致高潮？
7. 若连续分歧尾盘买核心，次日是套利思路还是格局思路？
8. 尾盘是否已抢筹，次日需求是否被提前消耗？

- 主线题材中“后排也可能是买点”的卡位筛选标准。
- 尾盘抢筹在不同情绪周期下，对次日溢价的实际影响。
- 离别钩的稳定识别条件和更多案例对照。
