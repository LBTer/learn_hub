## 当前体系地图

第一课确认，作者把系统划分为基础理解、量价和看盘、复盘工具、卖点与情绪周期、择时择股、交易心法及具体战法。作者明确认为战法次要，择时和情绪周期理解更重要。

当前已确认的基础结构：

```text
确认交易类型：投机或投资
→ 识别市场情绪位置
→ 判断资金从亏钱效应流向赚钱效应
→ 用胜率和赔率评价模式
→ 用小仓位完成试错
→ 接受单笔随机性
→ 重复长期正期望模式
```

后续课程需要把“情绪位置”“资金方向”和“模式有效性”逐步转化为可观察、可执行的规则。

第二课把其中的“资金方向”进一步落实为四组量价观察：

```text
顶部风险：高位爆量长上影 / 十字星 / 封死跌停
短期止跌：极致缩量与振幅收敛 → 阳包阴或连续小阳线
健康拉升：量价齐升，趋势未被明确信号否定
主力吸筹：放量阳线 + 缩量阴线，或连续小碎步阳线
```

当前课程依赖关系：

```text
第 1 课：理解情绪、资金和概率
→ 第 2 课：用量价读取资金行为
→ 第 3 课：用 K 线和支撑压力规范位置判断
→ 第 4 课：用复盘流程和工具沉淀盘面结构
```

第三课把“位置判断”拆成五组工具：

```text
K 线级别：日 K看结构，小级别 K线看日内细节
支撑压力：跌不下去是支撑，涨不上去是压力
箱体与支压转换：压力突破后转支撑，支撑跌破后转压力
趋势线与收敛：旧趋势失效后重新找箱体、趋势或三角收敛
均线与缺口：作为常用支撑压力，但个股上只能辅助
```

更新后的基础调用关系：

```text
第 1 课：先接受短线交易的情绪、资金和概率框架
→ 第 2 课：识别资金行为属于顶部、止跌、健康拉升或吸筹
→ 第 3 课：把量价信号放到支撑、压力、趋势和级别中验证
→ 第 4 课：每天按大盘、情绪、板块、个股和操作预期复盘
→ 第 5 课：把复盘输出放进盘中界面，跟踪资金流动、板块回流和风格对流
→ 第 6 课：用情绪周期区分混沌、主升、退潮，并匹配仓位和主线识别
→ 第 7 课：把情绪周期落到题材节奏表，识别试错、首次分歧、弱转强、龙头切换和退潮空仓
→ 第 8 课：在主流题材和赚钱效应里筛选领涨、带动、扛跌的强势核心票
→ 第 9 课：把核心票继续拆成情绪、身位、中军、弹性、容量、趋势、跟风和后排杂毛等角色
→ 第 10 课：把监管空间、严重异动和流速限制纳入择时与买卖点
→ 第 11 课：用小周期择时把混沌、主升、退潮落到买卖点
→ 第 12 课：用集合竞价提前观察方向、风险、核心票反馈和当天情绪
→ 第 13 课：在分歧或冰点活口中使用弱转强确认买点
→ 第 14 课：围绕确定事件提前低位埋伏，赚预期发酵到事件落地这一段
→ 第 15 课：把弱转强进一步细化为候选池、第一次转强、量能、套牢盘和赔率过滤
→ 第 16 课：把大成交机构趋势票细化为低吸买入、三日持股止盈、平台底部止盈和 CCI 辅助
→ 第 17 课：把题材节奏细化为强弱分歧、强弱回流、连续分歧修复、尾盘抢筹和一致回避
→ 后续课程：再细化补涨和其他具体战法
```

第四课把“每天要复什么”拆成一套工作流：

```text
大盘：技术分析 + 消息面 + 市场量能
→ 情绪：高标晋级、断板反馈、反包与 A 杀、情绪标的
→ 板块：节奏状态 + 龙头 / 情绪 / 容量 / 弹性 / 中军
→ 个股：涨停原因、概念变化、核心票池、热榜前 30
→ 操作：当天交易复盘、持仓预期、止盈止损
```

第五课把“盘中怎么看”拆成一套监控结构：

```text
竞价：一字方向 + 9:15-9:25 封单变化 + 竞价金额
→ 开盘：快速自然涨停是否呼应一字方向
→ 板块：涨幅榜看当天主攻，涨速榜看当下回流
→ 个股：短线精灵 + 三分钟涨跌速榜识别拉升、打压和出货
→ 反馈：核心票与后排是否一致，判断高潮、回流或分化
→ 对流：分时叠加和盘口漏单识别趋势、投机等风格 PK
→ 界面：自选核心、计划股、板块统计、盘口、L2、消息分屏观察
```

第六课把“市场环境”拆成三类情绪周期，并给出仓位和主线识别框架：

```text
混沌期：板块无序轮动 + 空间高度压制 + 小仓低位试错
主升期：主线明确 + 龙头带队 + 赚钱效应强于亏钱效应 + 重仓主线
退潮期：分歧后不能修复 + 周期龙跌停 / 高位补跌 + 空仓或极小仓博弈

主线候选：龙头 + 容量中军 + 20 厘米弹性 + 龙头属性发散
左侧发现：板块最强、新题材、强逻辑、持续消息
右侧确认：分歧扛跌、修复带头
```

第七课把“情绪周期如何实战”落到题材节奏表和节点动作：

```text
情绪只是外在，题材才是载体
→ 用题材复盘表归类涨停原因，标注身位、龙头、中军、容量、弹性、后排和核心
→ 混沌 / 退潮末期小仓做首板、一进二、二进三试错
→ 首次大分歧看后排是否仍有强度，低吸最核心、最扛跌的票
→ 弱转强或反包后进入主升二，主线题材提高仓位
→ 二次分歧后观察是否有龙头切换，能带板块的是周期龙，不能带板块的更接近妖股
→ 龙头跌停或主线接不住时按退潮处理，空仓优先
```

它补充了第六课尚未展开的实战节点：题材早期看启动日表态和身位票，首次分歧不只看前排断板，还要看后排强度；大题材可能通过龙头切换延续，小题材多数主升一、主升二就结束。

第八课把“择股”从题材节奏里独立出来，明确强势股筛选顺序：

```text
先择时和看主流题材
→ 在题材内找核心票
→ 用龙头三性判断：领涨、带动、扛跌
→ 用区间涨幅、竞价、涨速、赚钱效应和逻辑研究筛选
→ 再用个人模式过滤：中军、连板、低吸、埋伏、一字定方向各有不同指标
```

它补充了第七课提到的核心票特征：领涨可由区间涨幅排序发现，带动可由个股分时叠加观察，扛跌要看分歧日是否横盘、红盘或主动上攻；但作者认为最核心的是赚钱效应择股，即每个周期都要找当下最赚钱的图形或属性，再向类似票靠拢。

第九课把“核心票”进一步拆成复盘表里的角色定位：

```text
题材启动初期不一二板定龙
→ 先找情绪票：逻辑最硬、封单最大、能带动情绪
→ 再找身位票：先于节点启动，高一个身位
→ 用成交额找中军 / 容量
→ 用涨跌幅制度找 20 厘米 / 30 厘米弹性
→ 用走势识别趋势核心和加速风险
→ 用题材、代码、图形、名称识别跟风 / 补涨线索
→ 剩余后排用排除法降级为杂毛
```

它补充了第八课“核心思维”的具体落地方式：作者不主张普通交易者只做确认后的龙头，而是先把板块内绕不开的核心角色都选出来，等待市场通过领涨、抗跌、主动性进一步选择真正龙头。

第十课把“监管”从模糊风险拆成可计算、可套利但有阶段边界的变量：

```text
严重异动重点看 10 日 100% / 30 日 200%
→ 从当天往回拉窗口，取窗口前一天收盘价为起点
→ 起点涨停板抵消后，监管空间逐步抬高
→ 大票进入监管后受流速限制更明显，小票更多是情绪影响
→ 三类套利：监管限制前一天、绕监管最后一天尾盘、出监管前企稳
→ 只在强行情和监管套利赚钱效应存在时使用，退潮后失效
```

它补充了第七课情绪周期和第九课个股定位之间的约束条件：核心票能不能继续走，不只看题材和角色，还要看是否触发严重异动、资金是否害怕查账户、盘口大单是否会被流速限制压住。

第十一课把“择时”进一步落到大周期下的小周期买卖点：

```text
先定大周期：有周期龙是主升，没有周期龙是混沌，周期龙 A 杀是退潮
→ 混沌期：回流、高潮、分歧、分歧延续，高潮主卖，连续分歧后博弈回流
→ 主升期：仓位比个股重要，分歧给核心票买点
→ 退潮期：多数位置是卖点，只能少量博弈分歧延续后的反弹
→ 量化时代：小周期加速，必须放弃小分歧节点
→ 四类错误：分歧接面、主升空仓、冰点割肉、退潮追涨
```

它补充了第六、七课的执行层：情绪周期不是只用来分类，而是用来决定“今天该买分歧、卖高潮，还是干脆不做”。本课也明确了与第八课的边界：择时解决什么时候买，择股解决买谁。

第十二课把“开盘前怎么判断今天能不能做”落到集合竞价：

```text
工具：短线侠看 9:15 / 9:20 / 9:25 变化，同花顺做主界面，开盘啦适合手机
→ 看涨停封单：新方向、强方向是否减弱
→ 看跌停封单：反向一字定方向，识别空头宣泄
→ 看涨停加单：资金进攻态度，分歧后可能带板块回流
→ 看跌停加单：退潮和兑现风险
→ 看竞价金额：市场主攻方向，以及一致预期爆量卖点
→ 看昨天核心票反馈：龙头超预期转强还是不及预期瀑布
→ 输出当天情绪结论：利于开仓或不利于开仓
```

它补充了第五课和第十一课之间的执行环节：复盘选出的核心票要在竞价里接受反馈，竞价不是单独的买卖信号，而是开盘前给当天情绪和仓位积极度定调。

第十三课把“弱转强”作为具体战法接入前面的择时和择股框架：

```text
分歧日 / 冰点日的弱中带强
→ 核心、人气、辨识度票优先
→ 次日有修复或回流预期
→ 竞价带量高开，或低开后开盘第一分钟放量快速翻红
→ 竞价、翻红、半路、上板分层确认买点
→ 失败后纪律减仓或清仓，不做高潮次日和杂毛弱转强
```

它补充了第 7 课提到的“弱转强或反包后进入主升二”的具体执行方式，也把第 11 课的小周期择时落到单票买点。作者强调，弱转强不是独立万能公式，必须先有冰点、分歧、修复、核心票这些前置条件。

第十四课把“埋伏战法”作为适合上班族的事件驱动低吸模型：

```text
韭研公社时间轴寻找确定事件
→ 筛选时间确定、催化直接、板块容量大、辨识度高的事件
→ 用 AI 辅助找相关核心个股和逻辑
→ 提前 7-10 个交易日观察，事件落地前 3-5 个交易日低位买
→ 选择右侧微放量、有主力运作、未涨高、止损清楚的票
→ 发酵则分批止盈，事件落地清仓；破位则止损
→ 大盘下跌和短线退潮时少做
```

它补充了第 2、3、4、8、11 课之间的连接：复盘和逻辑研究先找到事件，量价和支撑压力决定能不能低吸，市场环境决定能不能做。作者反复强调，埋伏赚的是预期发酵的钱，不是事件落地后的右侧接力钱。

第十五课回到弱转强，把第十三课的基础买点进一步升级成一套进阶过滤器：

```text
候选池：昨日炸板 / 开板次数大于 3 的烂板 / 断板一天 / 热度榜辨识度未涨停
→ 位置过滤：尽量做第一次弱转强，第二次谨慎，第三次少做
→ 环境过滤：分歧次日、冰点次日、修复预期优先，高潮次日不做后排弱转强
→ 量能过滤：前一天爆量优于缩量，缩量转强要多观察
→ 筹码过滤：次日高开最好解放前方套牢盘，过不了前高不轻易打板
→ 赔率过滤：三到五个点或更温和的小高开优于八九个点极致高开
```

它补充了第 7、11、12、13 课之间的连接：弱转强不是“昨天弱、今天高开”的单点公式，而是题材环境、竞价反馈、筹码交换和盈亏比共同筛选后的交易机会。作者在本课特别强调，模式溢价下降后，能不能赚钱取决于审美和择时是否足够高。

第十六课把“机构趋势”作为一类大成交趋势主升模型，接入前面的量价、支撑压力和择股课程：

```text
活跃机构方向：AI 算力 / 半导体 / 新能源 / 创新药等
→ 股票池：逻辑研究 + 成交额排序 + 赚钱效应 + 热度榜
→ 图形前提：5 日、10 日、20 日均线多头排列
→ 买入：大阳线回踩实体一半，或关键均线低吸
→ 风控：买前设止损，阳线底部 / 均线 / 平台底部分别对应不同买点
→ 持有：单边主升用三日持股止盈法，平台洗盘用平台底部止盈
→ 辅助：CCI 站在上沿上方时提高持股耐心，跌回上沿下方时警惕卖点
→ 边界：大盘明显风险不逆势做，不套用到连板情绪小票
```

它补充了第 2、3、8 课和旁支趋势战法之间的连接：机构趋势不是单纯“涨得好”，而是成交额、机构方向、均线多头和趋势持有纪律共同成立。作者在本课特别强调，趋势的钱是熬出来的，但这个“熬”必须建立在预设止损、三日低点、平台底部和 CCI 等明确规则上。

第十七课把“题材节奏”进一步拆成可盘中确认的强弱分歧和强弱回流：

```text
核心票 + 后排 + 中军三类反馈
→ 强分歧：核心深水或跌停，后排区域跌停，中军负反馈
→ 弱分歧 / 分化：核心红盘震荡或不给机会，后排掉队
→ 强回流：核心反包前阴线，后排跟随，中军表态
→ 弱回流：核心冲高，后排冲高回落
→ 普通小题材：爆发、分化、大分歧的三天节奏
→ 潜在主线：分歧后强回流，给更多耐心
→ 交易动作：左侧买分歧，右侧买回流开始，回避一致高潮
→ 混沌期手法：连续分歧尾盘买核心，博弈次日修复
```

它补充了第 6、7、11、12 课之间的执行链：情绪周期和题材复盘不是静态分类，而要在竞价和盘中通过核心票、后排、中军的反馈不断修正。作者在本课特别强调，尾盘抢筹会提前消耗次日需求；一致高潮后的接力风险，往往来自需求减少和先手供给增加。

旁支专题“老庄趋势战法”暂不插入主课编号，而是作为一个具体战法模板，调用第 2 课的量价观察和第 3 课的五日线、趋势规则：

```text
十日 / 七日区间涨幅榜
→ 排除明确短线题材
→ 连红 K + 主力建仓痕迹
→ 发现日次日低吸
→ 收盘不破五日线持有，或越涨越卖锁定利润垫
```

它补充了“具体战法与基础体系如何连接”的一个样本：战法不是单独存在，而是先识别个股是否独立于短线题材，再用趋势顺畅度、五日线纪律和盈亏比处理买卖。

旁支专题“一字定方向战法”同样暂不插入主课编号，它把第 4 课的复盘表和第 5 课的竞价、涨速、盘中确认工具组合成开盘套利模型：

```text
板块启动第一天 / 第二天，或分歧后的二次转强
→ 竞价出现超预期大单一字
→ 开盘有后排自然涨停助攻确认
→ 选择最强换手、中军或弹性
→ 若无助攻、已一致高潮、前排给过机会，则放弃
→ 地板一字也可反向提示板块风险
```

它补充了“择时优先于战法”的证据：作者不是机械看一字，而是强调接分歧里的超预期，不接一致高潮；同一个板块过了第一时间后，后排套利逻辑会快速失效。

## 待验证问题

- 后续量价、K 线与情绪周期如何共同参与决策。
- 具体战法与基础体系之间是依赖关系还是独立模块。
- 独立老庄趋势与短线抱团之间的边界，是否能用题材标注和涨幅榜稳定区分。
- 一字定方向中“超预期大单”和“有效后排助攻”的量化阈值如何确定。
- 冰点次日修复预期下，弱转强的竞价量和开盘一分钟量能否形成更稳定的统计阈值。
- 第 8 课中竞价金额、竞价换手在分歧日和修复日的具体量化阈值如何确定。
- 第 12 课中涨停加单、跌停加单在不同情绪周期里的量化阈值如何确定。
- “预期”模型如何定义，才能支撑弱转强、回首掏和竞价反强的判断。
- 平台二波、反包、连板、N 字板等赚钱效应之间如何判断优先级切换。
- 逻辑研究选出的题材核心与盘面领涨核心不一致时，如何做取舍。
- 情绪龙、身位龙、中军、弹性、容量同时存在时，仓位如何在角色之间分配。
- 监管宽松期和严格期能否用公告、立案调查、停牌反馈等信号提前识别。
- 出监管前企稳的量价标准如何进一步细化。
- 监管套利与主线题材强弱冲突时，优先级如何排序。
- 分歧延续多少天、什么强度才算释放充分。
- 量化时代日内回流反杀后，隔日是否仍可继续博弈。
- 主升期“只接核心，不接后排”的仓位和买点细化。
- 计划成功率能否作为市场环境反推指标长期记录。
- 补涨和跟风里，题材模仿、代码模仿、图形模仿、名称模仿的优先级如何确定。
- 当前量化市场下，中军套利效应减弱后，哪些环境仍适合中军 / 容量套利。
- 作者如何统计各模式的胜率和赔率。
- L2 数据如何服务于不同的交易模式。
- “爆量”“极致缩量”和“微微放量”如何量化。
- 顶部信号在量化接力环境中的失效率。
- 主力吸筹痕迹能否稳定提高同题材择股胜率。
- 小级别 K 线与日 K信号冲突时如何确定优先级。
- 支撑压力允许多大偏差，才不算硬画线。
- 个股技术分析与题材、情绪、筹码、监管之间的权重如何分配。
- 板块节奏中分歧、分化、爆发、高潮、回流的细化判断标准。
- 强弱分歧、强弱回流和中修复的量化边界。
- 连续分歧尾盘买核心在退潮期、混沌期和主升期的胜率差异。
- 尾盘抢筹对次日需求和供给的影响能否通过竞价溢价统计验证。
- 主线分歧中后排卡位买点与杂毛反抽之间如何区分。
- 不同短线模式如何共用一套选股逻辑筛出核心票。
- 情绪标的的盘中确认方法，以及它和次日预案的联动方式。
- 盘中板块对流和互相吸血出现后，如何转化为具体仓位和买卖动作。
- 弱转强在量化套利较强的市场环境下，如何过滤冲高回落。
- 事件热度、催化直接性和板块容量是否能形成可复用的埋伏评分。
- 埋伏战法中 AI 选出的逻辑核心与盘面真正启动核心不一致时如何取舍。
- 大盘上涨、震荡、下跌环境对埋伏胜率和止损触发率的影响如何统计。
- 主升、混沌、退潮在事中如何进一步量化确认，避免把分歧误判为退潮。
- 龙头切换当天，如何区分有效接力和短线卡位套利。
- 分支题材弱转强在主线强势时如何给仓位和退出条件。
- 周期延续、复杂顶和反弹博弈如何量化确认。
