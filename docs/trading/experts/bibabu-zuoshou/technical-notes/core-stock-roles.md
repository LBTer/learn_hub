# 核心票角色分层

> 本文整理比巴卜作手关于板块内个股定位的跨期框架，不构成投资建议。个股仅作为方法语境中的角色示例，不能视为买卖建议。

## 框架定位

比巴卜不主张在题材启动初期就急着给唯一龙头下结论。他更强调“核心思维”：先把板块里绕不开的角色都找出来，再让市场通过领涨、带动、扛跌和主动性继续选择。

因此，复盘不是简单罗列涨停，而是把个股放进角色表里：

```text
题材启动 / 发酵
→ 情绪票
→ 身位票
→ 龙头候选
→ 中军 / 容量
→ 弹性票
→ 趋势核心
→ 跟风 / 补涨
→ 后排杂毛
```

## 主要角色

| 角色 | 核心观察 | 用途 | 风险 |
| --- | --- | --- | --- |
| 龙头 | 领涨、带动、扛跌三性更完整 | 代表题材高度和周期强度 | 事前难完全确认 |
| 情绪票 | 逻辑硬、封单大、买不到、能带动情绪 | 观察短线情绪和一致程度 | 开板可能就是分歧信号 |
| 身位票 | 先于节点启动，高一个身位 | 代表板块辨识度和空间 | 逻辑弱时容易被反推后淘汰 |
| 中军 | 成交额大、市值大、能承载资金 | 判断大资金态度 | 当前大成交环境下影响力可能稀释 |
| 容量票 | 成交活跃，板块绕不开 | 主线或趋势承接的重要锚 | 不等于短线情绪核心 |
| 弹性票 | 20 厘米、30 厘米或波动更大 | 前排买不到时观察套利 | 依赖板块情绪和后排强度 |
| 趋势核心 | 沿均线、平台或趋势推进 | 观察机构趋势和持有纪律 | 后半段加速风险上升 |
| 跟风 / 补涨 | 题材、代码、图形、名称等相似 | 情绪扩散和套利线索 | 空间和持续性通常弱于核心 |
| 后排杂毛 | 排除核心角色后剩余票 | 主要用于判断扩散质量 | 一致高潮后最容易兑现 |

## 龙头三性

核心票判断的底层仍是三性：

- **领涨**：涨得早、空间高、分歧日回流早。
- **带动**：上攻或上板时，板块后排跟随。
- **扛跌**：板块分歧时横住、红盘或主动上攻。

具备一两个特征，可以先放进核心候选；三性都更完整，才更接近龙头。但龙头不是主观封出来的，而是市场博弈出来的。

## 建立核心票池

比巴卜更现实的做法，不是普通交易者只做“龙空龙”，而是每个主流题材都先建立核心票池：

1. 先找逻辑最硬、封单最强、能带动情绪的情绪票。
2. 再找先于题材启动、高一个身位的身位票。
3. 用成交额和市值找中军、容量。
4. 用涨跌幅制度找弹性票。
5. 用走势找趋势核心和加速风险。
6. 用题材、代码、图形、名称相似性识别跟风和补涨。
7. 剩余票降级为后排，不作为重点交易对象。

这个流程的目的，是减少“每只涨停都想看”的噪音。真正要盯的是题材里绕不开、能影响板块次日预期的票。

## 角色会切换

同一板块内部，资金审美会变化。比如从情绪票切到容量票，从前排连板切到趋势核心，从高位核心切到低位补涨。比巴卜在复盘中经常强调，不能只问板块涨不涨，还要问板块内部是谁在涨、谁在掉队。

角色切换需要观察：

- 原核心是否出现负反馈。
- 新核心是否能带动板块。
- 后排是否跟随新核心。
- 中军或容量是否确认资金承接。
- 切换是良性的内部轮动，还是主线进入退潮的表象。

## 趋势与加速

趋势核心可以作为板块重要锚点，但趋势的后半段加速要格外谨慎。比巴卜会观察五日线、均线多头、量价、平台底部和 CCI 等辅助工具，但这些工具不能脱离题材和资金风格。

一个趋势核心如果开始量能缩小、斜率陡升、连续加速，往往说明风险收益比下降。分歧不一定杀死核心，后半段加速反而更容易带来见顶风险。

## 常见误区

- **一二板就定龙头**：题材初期应先找核心候选，而不是过早唯一化。
- **把所有连板都叫龙头**：身位高不等于有带动性。
- **中军等于板块节奏**：大成交环境下，中军影响力可能被分散。
- **弹性票强就代表主线强**：弹性更多是套利角色，要看前排和后排配合。
- **后排高潮后继续追**：后排最依赖情绪，一致后容易先兑现。
- **只按自己审美选票**：市场选择的核心，即使不符合主观喜好，也要尊重盘面。

## 失效边界

角色定位是动态的，不是贴一次标签就永久有效。情绪票开板、中军补跌、趋势核心破位、后排负反馈扩散，都会改变板块结构。

另外，核心票池不是买入清单。它只是帮助复盘和盘中观察的候选集合，真正交易还要回到周期、题材节奏、买点、仓位和失效条件。

## 复盘检查清单

- 板块里谁是情绪票，谁是身位票？
- 龙头三性中，哪些票具备领涨、带动、扛跌？
- 中军和容量票是谁，是否真的承接资金？
- 是否有弹性票或趋势核心值得观察？
- 后排是健康扩散，还是一致高潮后的兑现来源？
- 当前资金是在强化原核心，还是进行板块内部切换？
- 若核心候选竞争失败，具体退出或降级信号是什么？
