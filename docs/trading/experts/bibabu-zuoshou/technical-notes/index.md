# 技术点学习

用于跨多个交易日归纳比巴卜作手较稳定的交易方法、适用条件、常见误区和失效条件。

> 本模块只沉淀跨期反复出现的方法框架，不搬运单日观点，也不提供确定性买卖建议。单期视频中的判断仍应回到原始[交易日总结](/trading/experts/bibabu-zuoshou/trading-days/)和[视频文字稿](/trading/experts/bibabu-zuoshou/transcripts/)中追溯。

## 已沉淀专题

- [情绪周期与仓位](/trading/experts/bibabu-zuoshou/technical-notes/emotion-cycle-and-position)：把混沌、主升、退潮与仓位动作对应起来。
- [题材节奏、分歧与回流](/trading/experts/bibabu-zuoshou/technical-notes/theme-rhythm-divergence-return)：用核心、后排和中军反馈判断板块状态。
- [核心票角色分层](/trading/experts/bibabu-zuoshou/technical-notes/core-stock-roles)：区分龙头、情绪票、身位票、中军、容量、弹性、趋势和后排。

## 使用方式

阅读单日复盘时，可先用这些专题建立观察框架，再回到具体日期文章核对当时的指数环境、题材状态、核心票反馈和失效条件。若某个方法只在单期出现，暂时留在对应交易日总结里，不直接升级为技术点专题。
