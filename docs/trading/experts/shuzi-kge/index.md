# 数字K哥

这里整理 B 站 UP 主“数字K哥”的公开市场分析视频，重点保留其对大周期、短线结构、现货仓位、BTC/ETH 节奏和风险条件的判断。

> 页面内容是对公开视频的学习整理，不构成投资建议。视频中的判断具有时效性，应结合后续走势验证。

## 当前认知

数字K哥当前样本以币圈市场分析为主，常用“现货”“大饼”“以太坊”“五浪推动”“A/B/C 回调”“水平阻力位”“底背离”等语言组织观点。近期样本还显示，他会结合 ETH/BTC 汇率判断 ETH 相对强弱，用关键结构位区分正常回调与趋势反转，并把长期方向、短线进场顺序和试仓风险分开处理；判断突破质量时，还会交叉观察内部调整是否充分、MACD 动能是否延续以及成交量能否确认新高。

本页暂不固定其完整分析框架。后续积累多期样本后，再沉淀稳定方法、适用边界和失效条件。
