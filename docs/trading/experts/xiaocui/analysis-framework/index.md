## 待跨期验证的方法

### 经营流程反推护城河

2026-06-12 会员视频提出：研究陌生行业时，先假设自己从零经营该业务，把许可、土地、电力、人员、设备、客户、融资和回款逐步展开，再寻找不可快速复制的瓶颈。该方法能补足纯财务比率分析，但仍需同时检查瓶颈带来的交付风险、资本成本和替代路径。

### 杠杆与基本面分层
