## 三、作者给出的抄底条件

| 观察阶段 | 作者观点 | 条件/失效 |
| --- | --- | --- |
| 下跌趋势内 | 不急于抢最低点，当前抄底没有好止损位 | 继续下跌、支撑失守时不成立 |
| 脱离趋势 | 先脱离下跌趋势，再观察是否形成修复 | 重新跌回趋势线下方则需重新评估 |
| 回踩确认 | 回踩确认后，止损位置更清晰、空间更小 | 回踩不稳或继续放量杀跌则失效 |
| 底部形成 | 底部需要反复打压、磨底和确认，不会马上出现 | 不能用单日大跌或单日反弹确认底部 |

## 六、可迁移的单期学习点

1. **先看市场环境，再决定仓位。** 多指数共振下跌时，作者把“空仓”视为主动的风险控制动作，而不是错过机会。
2. **抄底需要结构和止损共同确认。** 不是看到跌幅大或缺口可能回补就参与，而是等脱离下跌趋势并回踩确认，以获得可执行的失效位置。
3. **量价关系要分方向比较。** 下跌放量、反弹量能不足，说明抛压尚未被反转；个股攻击套牢盘也需要持续放量，随后还要看回踩能否稳住。
4. **逆势强弱需要放到市场切换中复核。** 逆势上涨的医药板块在市场反弹时可能补跌，不能仅因当日强势就延展为持续性判断。

## 七、后续验证

后续应观察：指数是否脱离下跌趋势并回踩确认；缺口回补是否伴随趋势修复；作者列出的支撑位是否有效；放量攻击套牢盘的案例是否出现回踩稳住。以上仅用于回看作者条件是否触发，不构成对后市的确定性判断。
