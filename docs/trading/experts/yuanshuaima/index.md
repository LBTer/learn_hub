# 袁帅马

这里整理 B 站 UP 主“袁帅马”的个股趋势分析、主力操盘讲解、充电会员合集和方法观察。

> 页面内容是对视频的学习整理，不构成投资建议。视频中的判断具有时效性，应结合后续走势验证。

## 当前认知

袁帅马的普通公开视频目前更接近“个股趋势分析 / 主力操盘讲解”，不是标准交易日复盘。他会围绕某只核心个股的历史走势做详细解析，再抽取评论区或观众关心的个股，简单讲后续可以如何参与。

当前样本显示，分析重点包括：

- **趋势结构**：从历史走势中划分箱体、突破、回踩、洗盘、加速和逼空。
- **主力行为**：关注套牢盘消化、假头部、洗盘、对倒、换人、借市场弱势破位洗浮筹和诱多。
- **参与条件**：不强调直接追高，而是反复要求等回踩、找均线支撑、找盘中支撑、等止跌或等阴线修复。
- **风险边界**：缺口回补、趋势支撑失效、量能走坏、加速过度和上方套牢盘都是重要风险信号。

当前已积累多期样本，后续仍需要继续观察其稳定讲述顺序、常用指标和方法边界。
