# 疑问精研

这里记录从高手思路、每日复盘和交易学习中产生的疑问，并把不懂的地方逐步整理成自己的理解。

## 制作准则

每篇“疑问精研”优先采用“寓言式解释”来处理一个值得深入理解的概念：从相关领域中选取一个具有研究生水平、但确实能够被具体情境承载的概念，将它完整而间接地写进一则寓言（parable）中。

写作时应遵循以下结构：

1. 不要在标题、开头或故事推进过程中直接说出概念名称，也不要用术语逐段贴标签。
2. 通过人物、制度、资源、冲突、选择和结果，具体呈现概念的关键机制、适用条件与可能的误解；故事应自洽、完整，不能只做表面类比。
3. 将概念的正式名称和答案留到故事结尾，让读者先根据情节形成自己的解释，再在揭示处产生“原来如此”的顿悟。
4. 揭示之后附一段直白解释，明确说明：故事中的各个要素分别对应什么、概念究竟如何运作，以及这个概念在交易或现实问题中有什么用。
5. 如果寓言与真实概念之间存在边界或不完全对应之处，应在直白解释中主动说明，避免把文学类比误当成严格定义。
